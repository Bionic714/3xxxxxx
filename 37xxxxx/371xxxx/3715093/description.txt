Serial:
O serial ouro utilizado é visto como ORIGINAL e permite fazer quaisquer atualizações no site da Microsoft.

Observação:
Este CD inclui o SETUP.EXE que vem para atualização de um Windows antigo, porém, as intruções abaixo são para uma intalação Limpa [instalação após formatação do disco], pois esta elimina toda e qualquer possibilidade de uma instalação por cima de uma que já esteja comprometida por virus ou qualquer outro problema existente no sistema operacional anterior. Ele pode também fazer instalações por cima de um Windows 95, Windows 98 ou Windows 2000, exatamente como o CD original da Microsoft.

É recomendado a instalação LIMPA, ou seja, em um disco REFORMATADO.
O CD é "bootável" e deve ser gravado como IMAGEM ISO para que o sistema de BOOT seja ativo. Para gravar o CD basta ir no seu programa de gravaçao e pedir para "GRAVAR IMAGEM". Por exemplo, no Nero voce vai no menu "Gravadora" e pede o ítem "Gravar imagem", mostra a imagem ISO e ele grava um CD que dá BOOT.
Para que o BOOT seja do CD, voce deve ir na BIOS e pedir que o CD seja o primeiro na lista de boot.

HD SATA:
A princípio ele não instala em HD SATA, porém, os drivers do SATA estão no CD, só que o CD original da microsoft, não reconhece os HDs SATA, não é a minha versão.
Muita gente tem me dito que conseguiu instalar em SATA, mas eu não garanto a ninguém porque o CD original não tem esta capacidade, e como os drivers do SATA só são reconhecidos para a maioria das placas-mãe se inseridos com F6 e o disquete no inicio da instalação [antes do CD instalar os drivers] eu sei que tem a possibilidade de não instalar...
Voce pode tentar pra ver o que dá...

Atualizações:
As atualizações são adicionadas manualmente, e podem ser encontradas na pasta "1-ATUALIZAÇÃO". Siga passo-a-passo a instalação das atualizações, somente depois disso verifique se as atualizações foram corretamente aplicadas usando um dos programas que estão na pasta "Extras - Verificação".

Voce encontrará na pasta "2-EXTRAS" os programas Internet Explorer 7.0 e Windows Media Player 11, entre outros, e caso queira, poderá instalar todos ou nenhum.
Porém, não esquecendo do teu Anti-Virus preferido, caso não goste do Avast Pro que está incluso!!!
