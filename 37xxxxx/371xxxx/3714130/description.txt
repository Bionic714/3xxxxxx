Extract, then install the screensaver. Run the keygen and use the drop-down menu in the keygen to select "Mechanical Clock 3D". Use any registration name. Generate the code and copy it to clipboard. Under the screensaver settings in control panel you'll see registration. Enter the same name as use used in the keygen and then paste the code in. Enjoy!
