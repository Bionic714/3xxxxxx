
The minimum system requirements are as follows:
	
	OPERATING SYSTEM:
Microsoft Windows 2000/ Microsoft Windows XP/Microsoft Vista (admin rights required). 
CPU:
Pentium 4 1.7 GHz or Athlon XP equivalent 
RAM:
512 MB system Memory 
GRAPHICS:
ATI 9600/Nvidia 5950 or better (pixelshader 2.0 or higher)
SOUND:
Direct X 9.0c compatible sound card 
DVD-ROM:
DVD-ROM drive
HARD DRIVE:
2 GB Free Space 
INPUT DEVICES:
Keyboard and mouse

The recommended system requirements are as follows:

OPERATING SYSTEM:
Microsoft Windows 2000/ Microsoft Windows XP/Microsoft Vista (admin rights required).
CPU:
Pentium 2.5 GHz or Athlon XP/64bit Equivalent
RAM:
1 GB system Memory
GRAPHICS:
Pixel Shader 3.0 compatible card
SOUND:
Direct X 9.0c compatible sound card 
DVD-ROM:
DVD-ROM drive 
HARD DRIVE:
2 GB
INPUT DEVICES:
Keyboard and 3 button mouse
