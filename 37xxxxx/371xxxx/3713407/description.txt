The Ultimate Suicide Commando Collectors Item Box Set - This Limited Edition 3CD + DVD set includes a total of 50 tracks on 4 Discs including a Live DVD, a Remix CD, a Best Of CD PLUS the Exclusive "F*** You Bitch" Bonus Single ! X.20 also features an Extended Booklet with exclusive photo material from the band throughout the last 20 years ! This Box Set is Limited to 5000 copies worldwide. 

A high-quality constant in the Electro scene for 20 years : a hit machine, a DJ staple, and an audience favorite in every venue. Hits like "See You In Hell", "Hellraiser", or "Dein Herz, Meine Gier" have transformed SUICIDE COMMANDO not only into an Industrial icon worldwide, but also into a de facto label for of an entire genre of music. It's hard to believe that this Belgian EBM legend celebrated his 20th anniversary last year ( celebrated by a series of specially recorded Anniversary concerts which are included on this set ) - Additionally X.20 includes contributions from some of the biggest names within the Industrial scene with Exclusive Versions of Suicide Commando classics from SITD, Combichrist, Agonoize, Covenant, Spetsnaz, Wumpscut, Dope Stars Inc. and many more : Also included in this amazing set is a "Best Of" CD collecting highlights of the last 20 years ( selected by fans via Internet voting ) PLUS the long awaited first single ( "Fuck You Bitch" ) from 2006 album "Bind, Torture, Kill".  

None of the CDs included in this set will be released separately.
