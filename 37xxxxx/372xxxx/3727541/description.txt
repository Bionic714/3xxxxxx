Systeem eisen: 

Finale 2007 Windows
Windows 2000/XP. Garritan sounds alleen ondersteund onder Windows XP.
CD of DVD-ROM drive. 
800x600 minimum monitor resolution. 
Minimaal 256MB RAM (1GB of meer aanbevolen voor optimaal gebruik van Garritan sounds / Kontakt Player 2: meer RAM = meer beschikbare sounds). LET OP: Heeft u een on-board videokaart kan 256Mb RAM te weinig zijn om uw videokaart een deel van dit geheugen reserveert.
200MB harddisk ruimte benodigd voor software en user manual. Nogeens 800MB benodigd alle Garritan sounds.
