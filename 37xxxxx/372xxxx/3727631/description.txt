Foobar2000 with 3 best skins, tons of plugins and AlbumArtDownloader :D
It's silent, sfx archive. It will extract all files to "%programfiles%foobar2000", install fonts and add shortcuts to Start Menu. If you want update this archive, just open with WinRAR and copy your files.
----------------------------------------------------------------------------------------------------------------------
What's new in my build??:
- 3 cool skins (with different color versions)
- AlbumArtDownloader
- A lot of new, updated plugins (Winamp DSP bridge, Crossfader, Channel Matrix, send to/remove from IPod, AC3/DTS SPDIF, foo_Run and much more)
- WMP11 Visualization, Egoh Spectrum Analyzer and Peakmeter
- Miniplayer mode
- Possibility to select between ulimited backgrounds
- AlbumArt component for slideshows
- New Options panel in Playlist (sort order, playlist color, playlist selector, album list panel)
- New Options in Now playing panel (artist image, album cover, autoscale)
- New Options in Lyrics Panel (compressed lyrics, expanded lyrics, timestamped lyrics)
- Use Artist picture as background
- Use small artist image / small album cover
- Show/hide big album cover
- Button to get lyrics using lyricsDB
- Link to Artist website
- Rating buttons
- Artist images
- Album covers
- Black / White Playlist color
- Full Albums / Singles mode in Playlist
- PNG / JPG images format in Artist or Cover image
- Auto scale images in Now Playing panel
- Brightness / Shadow in settings panel
- Font and fontsize selection (in the main layout)
and much more...:)

Important notes:
Important notes:
1. Default installation folder is "C:Program Filesfoobar2000"
2. Front art should be called "*folder*.*" and back art should be called "*back*.*". You can also add "%artist%.*"
  Those files must be placed with album, e.g. in:
----------------------------
D:MusicMETALLICAReload
-TRACK1.mp3
-TRACK2.mp3
-...
-TRACK13.mp3
-folder.jpg
-back.jpg
-%artist%.jpg
-----------------------------
3. If someone have win 2000, try to install GDI+ http://www.microsoft.com/downloads/details.aspx?familyid=6A63AB9C-DF12-4D41-933C-BE590FEAA05A&displaylang=en
4. Install VC8 SP1 Runtimes. You can find it here: http://acropolis.lokalen.org/2006/10/foobar2000/my-foobar2000-components
5. You need Pentium 3+ or compatible computer, because some plugins use SSE instructions
