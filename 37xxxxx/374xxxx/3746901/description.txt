First burn the 4 images on 4 separate cd's, then follow the instructions below:

Installation:
1. Press: "NAVI/OK"
2. Insert "CD 1", choose: "UPDATE"
3. When "CD 1" is ready do a calibration trip
4. Afters calibration trip Insert "CD 2"
5. When "CD 2" is completed, youre ready with the update!

Good trip!

Europe is divided over 4 CD's:

"CD 1"  S / N / FIN / DK / GB / IRL
"CD 2"  B/ NL / L / D / DK / A / CH / FL / PL
"CD 3"  B / L / F / E / P / MC / AND / GBZ
"CD 4"  I / A / CH / FL / RSM / V / CZ / SK
