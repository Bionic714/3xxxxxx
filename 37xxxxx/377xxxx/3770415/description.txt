The GridinSoft Notepad is intended for those people who want more features than the standard notepad in Windows provides.

The basic feature of the program is syntax highlighting of more than 35 formats of files (HTML, ASP, PHP, Perl, Pascal, C++, etc.). Additionally the program features print preview, bookmarks, adjustable interface and many other features.
