GC/Wii Bongo Collection
***********************

This is a collection of all the games for Gamecube where you use the Bongo controller

The collection consists of the following games:

Donkey Konga (PAL)
Donkey Konga (USA)
Donkey Konga (JAP)
Donkey Konga 2 (PAL)
Donkey Konga 2 (USA)
Donkey Konga 2 (JAP)
Donkey Konga 3 (JAP)
Donkey Kong - Jungle Beat (PAL)

It is possible to burn these games on one multigameiso for Wii using the included tool. Read the readme for explanation of this.
