BitDefender Total Security 2008


BitDefender Internet Security covers all the security needs of an Internet-connected family. It provides comprehensive protection against viruses, spyware, spam, scams, phishing attempts, intruders and objectionable web content. BitDefender Internet Security is designed to place as little burden as possible on its users and on the host system, while providing state-of-the-art defense against current Internet threats.

New features included in BitDefender Total Security 2008
- New User Interface
- Tune Up and Backup
- New on-demand antimalware scanner
- Game mode
- New User Interface
