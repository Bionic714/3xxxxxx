Easy CD-DA Extractor - the "Swiss Army Knife of Digital Audio" is 4-in-1 software that rips Audio CDs, converts music files, and burns Audio CDs, MP3 CDs, WMA CDs, and Data CDs.
Easy CD-DA Extractor supports all popular audio file formats - 3G2, 3GP, MP1, MP2, MP3, Windows Media Audio, Ogg Vorbis, MP4, M4A, AAC, aacPlus v1 (HE-AAC, AAC+), aacPlus v2 (HE-AAC v2, HE-AAC+PS, eAAC+), FLAC, Musepack, WavPack, WAV, AIFF, Monkey's Audio, Shorten, CUE and M3U. Also, the Nero Digital (Nero AAC) is supported if you have the Nero Burning ROM installed on your computer. State-of-the-art audio encoding technology - Easy CD-DA Extractor includes the most advanced audio codec available, the aacPlus from the Coding Technologies. Additionally, Easy CD-DA Extractor also supports command line (.exe) encoders. Easy CD-DA Extractor compatible with Windows 2000 and Windows XP. Available in 26 languages.

Easy CD-DA Extractor actually contains four separate programs:
? Audio CD Ripper - Rips audio CD's.
? Audio File Format Converter - Converts audio files from one format to another.
? Audio CD Creator - Creates audio CD's from audio files.
? MP3/WMA CD Creator - Creates MP3 CDs, WMA CDs, MP3/WMA CDs, and any Data CDs.
