AmigaOS 3.9 x86 (Amithlon)


Contains:

Amithlon / AmigaOS XL ISO Image
SCSI and IDE Boot disk images for preparing x86 machine
Amithlon Updates
New Kernel



Amithlon is a modified version of UAE running on top of a Linux kernel enabling you to run AmigaOS 3.9 as close to native as you'll get on an x86 CPU - maximising the amount of resources available to the OS.

Also included is AmigaOS XL which allows you to run AmigaOS within the QNX real-time Operating System.
