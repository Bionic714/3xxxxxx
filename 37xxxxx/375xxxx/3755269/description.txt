1st Security Center es una potente herramienta de seguridad con la que puedes restringir el acceso a las opciones y funciones más importantes de Windows.

Con soporte multiusuario, te permite crear distintos perfiles de acceso con diferentes privilegios, de forma que puedes crear uno, por ejemplo, para que tu hermano pequeño o tus hijos jueguen con el PC sin temor a que hagan algún estropicio.

Entre otras cosas, te permite bloquear el acceso al Panel de Control o a las aplicaciones de DOS, impide la edición del Registro de Windows y bloquea el acceso a Internet. También incluye numerosas opciones de configuración de seguridad para el Internet Explorer.

Además, puedes limitar el tiempo de uso del ordenador, estableciendo una especie de horario distinto para cada usuario. El programa guarda un log con todas las actividades realizadas en el PC por cada usuario y protege con contraseña las opciones de administración
