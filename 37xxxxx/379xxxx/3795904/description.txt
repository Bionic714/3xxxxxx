Motorola USB Driver 2.9.0 for Vista 32Bits


This USB Driver is used for connecting your Motorola handset to a 32-bit Windows computer via a USB cable.

Supported Handsets
This driver supports most Motorola OS, Motorola Linux OS, Motorola iDEN, Motorola Symbian/UIQ, Motorola Windows Mobile┬« and Motorola i-Mode handsets.

System Requirements
A 32-bit version of Windows 2000, Windows XP or Windows Vista OS 
You must have Administrator permissions to install this driver 
Notes:
Please reboot the system after installing this USB driver
Installation of Motorolas Mobile Phone Tools may affect the performance of this driver. Please re-install the USB driver after the installation of the Motorola Mobile Phone Tools
