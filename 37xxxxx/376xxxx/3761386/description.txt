This ISO file contains a menu driven Installation package with keygens for a host of Programs.  Some of these are AutoCad 2007, MS Office 2007, Adobe Photoshop CS2, Ashampoo, Tuneup Utilities and much more.

For the Newbie (newer than me), simply use Nero to burn to a disk.  Put hte disk back in and Autoplay will bring you a menu.  Enjoy.  This work is a compilation of many other peoples work and they are all credited for their work.  I simply compiled all of these into an easy to use Menu driven file.
