This is a compilation of all the install disk for Final Fantasy XI original through Aht Urghan for the PC. If you have an account on 360 or ps2 then you should be able to login on this pc version. There are no cd keys provided. You need an active PAID account for these discs to work. Just run the setup in the folder that you need. The disks should be installed in numbered order starting with Playonline. This package is primarily for users with lost broken disks. They are not in ISO/Image format. They are just full DVD/CD Dumps. Open the directory and run setup. Easy.
