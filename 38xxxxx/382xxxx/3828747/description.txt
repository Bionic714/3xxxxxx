INSTALLATIONSHINWEISE: ********************** 1.) Alte Version deinstallieren 2.) PC neu starten 3.) Setup Nero-8.1.1.0_deu_trial.exe ausführen 4.) Sollte die Meldung: "Ihr System wurde nicht verändert. Um Nero zu einem anderen Zeitpunkt zu installieren, starten sie das Setup erneut" erscheinen, bitte das Windows Update aktivieren. Dies geht unter Start => Einstellungen => Systemsteuerung => Leistung und Wartung => System => Reiter Automatische Updates => Automatisch aktivieren. 5.) PC nun neu starten 6.) Nun wieder das Setup Nero-8.1.1.0_deu_trial.exe ausführen (sofern es nicht automatisch nach dem Neustart neu startet) 7.) Serial/Seriennummer: XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX (die andere angezeigte Serial bitte löschen. Ansonsten gibt es nur eine 15 Tage DEMO Version!) 8.) PC neu starten 9.) Windows Update wieder deaktivieren unter: Start => Einstellungen => Systemsteuerung => Leistung und Wartung => System => Reiter Automatische Updates => Automatisch Updates deaktivieren 10.) Im Nero Startcenter rechts unten auf das gelbe Icon klicken. Im neuen geöffneten Fenster links auf Updates klicken. Alle xx Tage nach Updates suchen hacken entfernen. Mit OK bestätigen 11.) Das wars.
