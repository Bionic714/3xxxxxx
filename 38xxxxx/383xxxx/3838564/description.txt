Det här är Microsoft Windows 98 SE Swe. Om du har några frågor om detta operativsystem, så kan du fråga mej! ;) 
Jag har ingen snabb uppladdninghastighet, bara 50KB/s.
Så här gör du för att installera Microsoft Windows 98 SE:


Först måste du bränna Microsoft Windows 98 SE på en CD.
Det kan du göra med dom flesta brännarprogrammen som finns ute idag.

Sen skall du starta datorn med CD-skivan Microsoft Windows 98 SE.
Om inte det funkar, så kan du ställa in i BIOS:et att datorn skall starta ifrån CD-ROM.
Du kan även testa att ladda ner en startdiskett om du inte vet hur man ändrar i BIOS:et.
Du kan ladda ner en startdiskett här: http://files.frashii.com/~bootdisk/newjersey/boot98se.exe

Starta sedan installationsprogrammet.

Produktnyckeln som du behöver finns i textfilen, Produktnyckel.txt


OBS! Ett tips, installera på en nyformaterad hårddisk, då borde det inte uppstå några problem.


Med den här CD'n kan du installera ifrån tidigare Windows-versioner.


Den här fungerar både i en vanlig dator och i en Virtual PC. Jag har testat med Parallels Workstation, VMware Workstation och Virtual PC 2007, funkar perfekt!


Jag vet inte om den funkar att bränna på DVD, men jag tror att det skall fungera.
