Esto es un port del juego Baldur's Gate II Shadows of Amn y su expansion Throne of Bhaal de Windows hecha con Cider para Mac con procesadores Intel.

This is a Cider port for the Windows game Baldur's Gate II and the expansion pack throne of Bhaal, it only works on Intel macs.



Requisitos (segun la Wikipedia)

Intel Pentium II 233Mhz
32MB Ram
4MB Video RAM
2.7GB Hard Disk


Probado en: / Tested on: 

Macbook 
Intel CoreDuo 2Ghz
512 RAM
Intel GMA 950
Mac OS X 10.5 (9a559)


(Funciona con Mac OS X 10.4.8 o posterior / Works with Mac OS X 10.4.8 or later)
