With "Crush On Hardcore 1" we had alot of problems, this time we did it all better.

Harry H, Mr E, Ruffage, Bounce & Square.

In the .rar files you can find: a 320kbps mp3 file of the mix + a .cue file to split the mix.
You can extract the .rar files with a program called Winrar.
If you would like to split the mix up with the .cue files, you can easily do that with a program called Cue Splitter.
