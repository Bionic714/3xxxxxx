BAI (Blat Auto Install) is a simple console program.

It will facilitate a default Blat install;  
http://blat.net
Blat help; 
http://tech.groups.yahoo.com/group/blat/
More Blat files; 
http://tech.groups.yahoo.com/group/blat/files/Other/

Double click on bai.exe to run. Follow the on 
screen prompts.

BAI currently supports Outlook and Outlook Express.
If both are found it will install from Outlook.

BAI.exe is self contained. No other files are required.

The source folder has the files needed to run bai.pl.
