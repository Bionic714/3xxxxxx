This is it. Brazil's busiest airport has been redesigned and is now available for both FS9 and FSX. Using latest rendering tecnologies, the new version of Congonhas will give you a real feeling of being there. The scenery is available in two seperate installers, one for each simulator. 

FEATURES

Carefully detailed 3d objects 
Full night illumination 
Compatible with default FSX scenery 
Compatible with add-on mesh. 
Compatible with add-on landclass 
Compatible with add-on AI traffic. 
Usage of the new FSX SDK for FSX version and FS9 SDK for FS9 version 
Auto-installer
