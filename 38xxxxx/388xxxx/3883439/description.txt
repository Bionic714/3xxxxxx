ReturnOfNights Mega Game Pack (v2)
Screen-shot: http://img105.imageshack.us/img105/7841/mgp2tc1.jpg

Here's the 2nd Pack of 187 Little games to fill in your spare time with.

Each of the Games has been patched to remove annoying 60 minutes remaining 
screens, all the crap removed and no need to install them as they don't require 
registry entries.

The pack includes an Installer/Launcher that enables you to see categories, 
screen shots and descriptions for each game. It also creates a start menu 
shortcut in the Categories under the path specified in 
'X:\WPIScripts\startmenu.js' so feel free to adjust this to suit your needs 
BEFORE burning off the ISO -- as this is a non bootable disk it is easy to 
modify, no need to worry about breaking it (unless you leave something off).

The Launcher is stored in 'X:\Install\GameLaunch2.exe' so if you add games then 
this will need to be updated as well as the Installer, BUT if you remove a game 
folder from the installer then it will not be shown in the selection menu.
