~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
~Install~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. Run pacsteam-03-10-07.exe or PacSteam-24-10-07.exe.

2. Copy PST-TEMP.7z to X:PacSteamT (update auto downloading this file).

3. Update PacSteam.

4. Run shorcut PacSteamT and please wait update Steam Client (to be possible long time or fast...:)()

5. After update Steam Client is to completed, you to create account (free100%) and login with this account, in Steam/My games click right on game you need to choose Install game...

6. If you to receive message Steam client current are busy... turn off Windows Friewall and another, go www.steampowered.com.

7. Enjoy.
