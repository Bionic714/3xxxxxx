     |                                                                    |
     | Sinhá Boça é uma novela que foi exibida na MTV Brasil, que         |
     | estreou no dia 21 de novembro de 2006, às 22:00 e teve dia 6       |
     | de dezembro de 2006 seu último episódio, um especial com uma       |
     | hora de duração.                                                   |
     |                                                                    |
     | -----------------------------x------------------------------------ |
     | Enredo:                                                            |
     | Sinhá Boça conta a história de Boça - O Paulistano -, um sujeito   |
     | simples e simpático, porém azarado e que vive uma vida difícil.    |
     | Ao contrário de seu pai, que é um bom malandro, Boça é esforçado   |
     | e concilia seu trabalho de office-boy com os estudos na faculdade  |
     | e os jogos de fliperama.                                           |
     |                                                                    |
     | Ele se considera responsável por uma empresa na qual é apenas um   |
     | office-boy e foi eleito o melhor durante dois anos consecutivos.   |
     | Ele vive com a avó D. Lurdes, "coroinha maneira" que faz de tudo   |
     | para agradar netinho criando-o "a base de leite com pera,          |
     | ovomaltino e doces caramelados na geladeira."                      |
     |                                                                    |
     | Um dia indo para sua "facul", como chama, ele vê Suzana e logo se  |
     | apaixona perdidamente por ela. Mas ela namora Jubão, um playboy do |
     | qual Boça vivia apanhando.                                         |
     |                                                                    |
     | Do outro lado há Rafa, o chapeiro que vende lanches na porta da    |
     | faculdade. Rafa sonha em se tornar um astro da música e compôs o   |
     | sucesso "Boladão de Amor". Rafa namora Jaqueline, que trabalha de  |
     | doméstica. Ela é humilhada pela patroa.                            |
     |                                                                    |
     | Boça foi para a cadeia acusado de roubar o "Lápis de Ouro", mas na |
     | verdade foi Jubão que o colocou na mochila de Boça.                |
     |                                                                    |
     | O julgamento é decidido por uma briga entre Boça e Jubão e quem    |
     | ganhasse seria condenado. Boça acaba levando uma surra e perde a   |
     | luta. Ele volta para a cadeia mas acaba fugindo no final da novela |
     | - "agora tudo faz sentido!".                                       |
     |                                                                    |
     | O trovadorismo em 'Sinhá Boça':                                    |
     | O amor platônico de Boça por Suzana é baseado no trovadorismo da   |
     | idade média, quando princesas se casavam com príncipes e assim os  |
     | servos que as amavam teriam que ficar calados ou iriam morrer. No  |
     | caso de Boça, sua princesa Suzana namora com um playboy, só que    |
     | como Boça não se cala, leva porrada todo episódio.                 |
     |                                                                    |
     | -----------------------------x------------------------------------ |
     | Personagens:                                                       |
     | Luis Boça é o protagonista da novela. Aluno de uma faculdade de    |
     | Direito em São Paulo, tem uma paixão por Suzana, sobrinha de       |
     | D. Máxima Cantarola e namorada de Jubão. É interpretado por        |
     | Felipe Torres.                                                     |
     |                                                                    |
     | D. Lurdes Boça é a uma "velhinha guerreira" que criou seu neto,    |
     | Boça, a "base de leite com pêra, ovo maltino e pão com mortandela" |
     | (definições dadas pelo Professor Gilmar, de Direito Penal). É      |
     | interpretada por Marco Antonio Alves.                              |
     |                                                                    |
     | Máxima Cantarola é uma vilã perversa que adora maltratar a sua     |
     | empregada Jaqueline, obrigando-a, inclusive, a cortar um de seus   |
     | dedos por ter queimado sua blusa. No final da novela, descobre-se  |
     | que ela é uma lobisomem. É interpretada por Fausto Fanti.          |
     |                                                                    |
     | Rafa é um chapeiro, "excelente culinário" que faz um ótimo sanduíche
     | que "não dá pra comer só um". Ele conhece um grande produtor musical,
     | Manuelzinho Araújo e quer lançar seu hit, Boladão de Amor. Namorado|
     | de Jaqueline, pretende fazer sucesso com sua música pra poder tirá-la
     | de D. Máxima. Teve "vários confrontos de ego" com o grafiteiro Cuta.
     | É interpretado por Bruno Sutter.                                   |
     |                                                                    |
     | Cuta é o grafiteiro que dialoga com sua família cantando hip-hop.  |
     | Conhece o artista de pop-art Alec Le Bleur que faz mil promessas que
     | iria fazer dele um artista renomado mas o que o artista queria (e  |
     | consegue) é dormir com o grafiteiro. É interpretado por Adriano    |
     | Pereira.                                                           |
     |                                                                    |
     | Jaqueline, empregada de D. Máxima, é maltratada e vive como uma    |
     | escrava. Tem que pagar o aluguel de seu cubículo e até apanha no   |
     | tronco. É interpretada por Marco Antonio Alves.                    |
     |                                                                    |
     | Suzana, sobrinha de Máxima e namorada de Jubão, é a garota mais    |
     | bonita da faculdade. É interpretada por Bruno Sutter.              |
     |                                                                    |
     | Jubão, playboy badboy da faculdade, adora bater em Boça. É         |
     | interpretado por Marco Antonio Alves.                              |
     |                                                                    |
     | Prof. Gilmar, professor de direito penal na faculdade de Luís boça,|
     | possui um caráter ilustre, letrado, cavalheiro, com toques da mais |
     | fina etiqueta e galanteios sofisticados. Muito influente na OAB    |
     | (Ordem dos Advogados do Brasil), trabalhou em diversos casos nos   |
     | Estados Unidos, tendo como maior triunfo, a vitória na defesa de   |
     | O.J. 'Uól' Simpson. Carinhoso com suas amantes, vive a problemática|
     | do concílio entre trabalho e amor, varando madrugadas no seu Lap Toc.
     | No seu iluminado exercício do magistério, tenta por vias incisivas,|
     | transformar os alunos em cidadãos competentes e corajosos, sendo   |
     | algumas de suas frases: "Tudo os aluno criado a leite com pera,    |
     | a ovomaltino, a pão com mortadela!", "Não guenta dez minuto de     |
     | porrada comigo!". Tido como um dos personagens mais hilários da    |
     | mini-novela, é ele quem apresenta a Rafa; o chapeiro, um renomado  |
     | produtor do ramo musical para lançar seu hit "Boladao de Amor" acaba
     | defendendo Boça durante o julgamento. É interpretado por Gil Brother.
     |                                                                    |
     | Manuelzinho Araújo, empresário apresentado ao Rafa pelo Dr. Gilmar,|
     | o professor de Direito. Ele passa a perna no chapeiro aspirante a  |
     | funkeiro pedindo enorme quantia em dinheiro, dando a justificativa |
     | que iria ser utilizado para lançamento de CDs, shows, jabá para as |
     | rádios, etc. Prometeu ao ingênuo Rafa que ele iria se tornar o     |
     | Príncipe do Funk mas ao invés disso ele rouba o seu hit Boladão de |
     | Amor e empresaria um outro artista, MC Dande da Barra da Tijuca,   |
     | que se se torna o intérprete da música criada pelo chapeiro. É     |
     | interpretado por Fausto Fanti.                                     |
     |                                                                    |
     | Alec Le Bleur, artista plástico francês que se encanta com o pichador,
     | opa, grafiteiro Cuta e o seduz levando para cama. É interpretado   |
     | por Fausto Fanti.                                                  |
     |                                                                    |
     | Ciro Pantera, professor de Criminalística é também o guardião do   |
     | troféu Lápis de Ouro.                                              |
     |                                                                    |
     | Acadêmicos são interpretados por fãs do quinteto Hermes e Renato.  |
     |                                                                    |
