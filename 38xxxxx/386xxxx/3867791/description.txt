Burning to Dual Layer DVD:
* Burn the ?Mac OS X Install DVD.dmg? to a dual layer DVD.
* Insert the DVD into the computer
* Restart and on startup maintain the [C] key press the until it boots the DVD

Copying to an external USB drive:
WITH DISK UTILITY
* Create a 8GB partition with MacOS9 drivers
* Restore the ?Mac OS X Install DVD.dmg? to the partition
* Leave the external hard drive connected and restart your computer
* On startup maintain the [OPTION] key until you see the startup drives
* Select the ?Mac OS X Install? option

Good luck
