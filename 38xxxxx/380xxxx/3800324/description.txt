Say goodbye to coding XML and generating images for SlideShowPro by hand. SlideShowPro Director is built from the ground-up to fully administer one or more slideshows created with SlideShowPro. Completely browser based and chocked-full of Ajax interactivity, Director is the quickest way to get your images into SlideShowPro. It generates all large images, thumbnails, provides image captioning inputs, manages albums, sorts images, publishes XML data, and generates dynamic galleries.
