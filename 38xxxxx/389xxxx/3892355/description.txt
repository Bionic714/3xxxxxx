This is the Daedalus R13 Nintendo 64 Emulator for use on the Sony PSP. This requires 1.5 firmware or the ability to play 1.5 homebrew, e.g. (3.40OE, 3.52 M33, ect...) This cannot be played on any *official* sony firmware above 1.5.

To install unzip the Daedalus zip file and put the Daedalus & Daedalus% folders in your PSP/GAME or PSP/GAME150 folder on your memory stick.

(The Daedalus R13 emulator now has a Save State feature!)
