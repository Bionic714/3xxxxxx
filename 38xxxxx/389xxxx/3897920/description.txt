_________________________________________________________________
-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_- 1.1 MINIMUM SYSTEM SPECIFICATIONS

LEGO Racers 2 requires the following system specification as the 
absolute minimum, for the game to function correctly and at a 
playable level.

 - PROCESSOR        >	Pentium II 266MHz

 - MEMORY           >   64MB System RAM

 - VIDEO            >	16MB Display Memory / Direct3D Compatible

 - SOUND            >	100% Windows(R) 95/98/ME - DirectX(R) 8.0
			compliant.

 - CD-ROM           >	12X or Greater CD-ROM / DVD Drive

 - HARD DRIVE SPACE >	200MB / 450MB [Free and uncompressed*]

 - INPUT DEVICE/S   >	Windows 95 / 98 / ME - 6+ Button-Joystick
			6+ Button-GamePad and Keyboard.

 - DIRECTX          >	LEGO Racers 2 requires the installation 
			of DirectX(R) 8.0a
			[This is provided on the Racers 2 CD....]

* - Uncompressed means that the hard drive space has not been
    subjected to compression methods to increase capacity.

Note: If your system does not conform to the above minimum
      requirements, LEGO Racers 2 will not function correctly on
      your system.

-_-_-_-_-_-_-_-_-_-_-_-_-_- 1.2 RECOMMENDED SYSTEM SPECIFICATIONS

LEGO Racers 2 contains some stunning visual effects and high
levels of detail.
The following specification is recommended for optimal
performance and visual quality while playing LEGO Racers 2.

 - PROCESSOR        >	Pentium II 400MHz

 - MEMORY           >   128MB System RAM

 - VIDEO            >	32MB Display Memory / Direct3D Compatible

 - SOUND            >	100% Windows 95 / 98 / ME - Direct-X 8.0 
			compliant

 - CD-ROM           >	16X or Greater CD-ROM / DVD Drive

 - HARD DRIVE SPACE >	450MB [Full Installation Option]

 - INPUT DEVICE/S   >	Windows 95 / 98 / ME - 6+ Button-Joystick
			6+ Button-GamePad and Keyboard.

 - DIRECTX          >	DirectX 8.0a

-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-  2.1 OPERATING SYSTEM

LEGO Racers 2 has been designed to operate on Microsoft Windows
95/98 and Windows ME. The game has been tested for functionality
on the following operating systems:

	> Microsoft(R) Windows 95A [Retail]
	> Microsoft(R) Windows 95B [OSR/2]
	> Microsoft(R) Windows 98 First Edition
	> Microsoft(R) Windows 98 Second Edition
	> Microsoft(R) Windows Millennium Edition

LEGO Racers 2 will also function on the following operating 
systems:

	> Microsoft(R) Windows 2000 Professional [NT5]
	> Microsoft(R) Windows XP

The game was not designed to run on the following operating
systems:

	> Windows(R) NT4 OR LOWER
	> OS/2
	> Linux
	> Emulated Windows Environments
