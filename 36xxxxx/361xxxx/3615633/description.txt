Date: February 10, 2007 20:11:00

Description:
mIRCPatch.exe removes the following
- CRC/Filename check
- VERSION reply


Installation:
Run mIRC621.exe to install mIRC
Extract SPK-mIRC621Patch.zip to mIRC folder
Execute mIRCPatch.exe and patch mIRC.exe
Open mIRC
Type in the input bar... /load -rs miRCKeygen.txt
Navigate to Commands->Register mIRC
