Jasob JavaScript Obfuscator protects your JavaScript code from stealing and significantly shrinks size (for around 70%) to improve download time and execution speed in the Web browser. It is capable of reading JavaScript code from the various sources including pure JavaScript Files, HTML, ASP, ASP.NET, PHP, JSP, XML, Cold Fusion, ePerl; and you can define your own file types. JavaScript names that can\'t be changed in the process of obfuscation as well as rules for JavaScript code blocks for various sources can be customized. When opening files, it can open a single file, multiple files, the project or the whole Web site. There is the file viewer that shows both source and obfuscated file contents including tools for analysis, as well as there is support for names that appear in JavaScript strings and support for dynamic Web pages.
