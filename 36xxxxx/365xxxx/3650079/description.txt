-------------------------------------------------------------------------------
                                 Release Notes
-------------------------------------------------------------------------------
This is a heavily cut down version of Photoshop that can be run from a USB drive 
or another portable medium with ease. Please note that everything is packed with 
UPX which may set some Anti-virus programs off.

-------------------------------------------------------------------------------
                                 Install Notes
-------------------------------------------------------------------------------
1. Run PortaShopCS2.exe to self-extract.
2. Run either PhotoshopPortable.exe or ImageReadyPortable.exe.
3. Enjoy!
