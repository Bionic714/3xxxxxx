Spanish Win XP based on TinyXP by eXperience
Win XP Español basado en TinyXP de eXperience

PequeXP ha sido creado para ser una instalacion pequeña y rapida, para jugadores o aquellos que necesiten exprimir al maximo su PC. He eliminado muchas funciones y accesorios de windows para optimizar el sistema; el IE, pero he dejado el nucleo para poder usarlo con firefox, Opera o actualizar al IE7. No estan tampoco el Messenger, Media Player 10 (pero si el 6.4), Outlook, etc. Esta instalacion es para aquellos que prefieren usar otros programas mas concretos para cada necesidad. Tambien aquellos con ordenadores antiguos y poco potentes se benefician con esta instalacion. Y siempre queda el recurso de volver a instalar programas de windows con el CD original de instalacion de windows XP que debeis tener.

Esta version de Windows XP esta basada en la version inglesa de tinyXP de eXPerience (2006) y ha sido extraida de una version en español de Windows XP Pro Corporate con SP2 con ayuda de nlite. He dejado todo el soporte de hardware asi como la conectividad en red e inalambrica. Pero he eliminado todos los drivers de terceros. Antes de instalar comprobar la posesion de los drivers correspondientes. Leed el archivo de instrucciones incluido en la ISO.
La instalacion es completamente silenciosa. Desde el momento que empieza a formatear la particion donde se instala ya no solicita ninguna informacion.

Version de Windows XP realizada con nLite  http://www.nliteos.com/
El archivo Ultima Sesión.ini contiene los parametros de esta version

Para algunos sistemas (C2D + 650i al menos)esta ISO debe ser grabada en un DVD (yo uso +RW sin problemas)aunque solo tenga 154Mb. Esto es a causa de un bug en nlite que aun no ha sido resuelto.
