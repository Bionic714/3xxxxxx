This is a custom made DragonFly BSD 1.8.1/i386 Live and install CD

It differs from the official release as it contains:
* DragonFly user manual
* Full sources
* DragonFly logotype images

How to burn this CD:
1. Unpack DragonFly_BSD-1.8.1.iso.bz2
   You may use 7-zip (www.7-zip.org) on the Windows platform
2. Burn DragonFly_BSD-1.8.1.iso with your favourite CD burning program.
3. Boot from the CD
