Con este pack podrás instalar los programas en varios PC's y crear tu propio cyber café. Con el servidor podrá controlar los demás ordenadores y configuarlos para los juegos y programas que puedan utilizar. Podras tener total control, y usar un sistema de tickets y usuarios para éste. :)

With this pack you can install the software in some PC's and create your own Cyber. With the server you will control the rest of Pc's and config as you like. In addition you can create a ticket & user system. :)
