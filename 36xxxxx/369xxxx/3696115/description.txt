A Generic steam patcher. Works with all versions of the steam client. 

1. Download the steam client: http://steampowered.com/download/SteamInstall.msi

2. Install, create account if necessary.

3. Place patcher in the steam directory where steam.exe is and run it.
   (steam must be turned off)

4. Enjoy

All games should show up. not all games are playable or downloadable. Try to search for gcf files.

And no. It\'s not a virus :)
