S.T.A.L.K.E.R. Multi Launcher (aka. S.T.A.L.K.E.R. Multiplayer Tool) information: 

With the S.T.A.L.K.E.R. Multi Launcher Tool you can:
  - Join to a S.T.A.L.K.E.R. server
  - Create an own server and join to it
  - Create a dedicated server
  
You can easily configure your server just by a few mouse clicks.
The program saves your settings and you don't have to re-set them at next time.
This tool supports multiple languages by language files and it's easy to create  
your own language file just see the included file for the syntax.                                                                                  

Version: 1.2.0.2
Release date: 2007.05.28.
Web: http://tomikaa.uw.hu

The pack contains:
 - sml.exe - S.T.A.L.K.E.R. Multi Launcher Executable
 - hungarian.lang - Hungarian language file
 - french.lang - French language file
 - spanish.lang - Spanish language
 - S.T.A.L.K.E.R.Multi.Launcher-ToMikaa.nfo The NFO file that contains the
   installataion how-to and the changelog
 - Lisez_moi.txt - French readme file
 - Leeme.txt - Spanish readme file
 
* Added CD-Key change and generate function to avoid CD-Key validation problems
* Added feature of creating maprot_list.ltx to add all maps to the server
