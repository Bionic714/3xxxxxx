== About DVD Decrypter ==
DVD Decrypter is one of the easiest DVD ripping tools you could get. With alot of settings, there was no way you couldn't barely ripp a DVD.

DVDdec ripps in 3 modes;
VOB mode (you can extract only the movie or the episodes of a series in this mode)
FILE mode (you extract the DVD with video_ts and audio_ts intact)
ISO mode (you ripp the DVD, or CD, into a ISO Image format)
