This is a .cdr file which contains Reign of Chaos for Mac OSX. Please keep in mind that this is an earlier version, and as such will require patching for certain functionality. CD key not included, but you can find one at the following URL: http://cracks.thebugs.ws/

!Important!
In order to use the .cdr file, follow these instructions:
1. Right-Click the file and select Open With -> Disk Utility
2. Select Warcraft III Mac OSX and click "Burn" in the top-left corner with  
   cdr inserted
3. Once the cd is finished, re-insert the cd and run installation
4. You're done!
