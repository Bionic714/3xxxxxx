OCMan Live XP ??? Késako?

RAPPELS DU OCMan BOOT CD : 

La version 2 de OCMan Boot CD FR est enfin arrivée!!!

Beaucoup de changement dans cette release, le principal étant la création complete du XPE et non des modifications d'un minipe existant ( meme si le XT-Pe etait excellent, sa vieillesse en faisait un mauvais support pour les nouvelles configurations.)

L'OCMan Boot CD est un CD de déplantage diagnostic qui je l'espère sera utilisé par le plus grand nombre d'entre vous, amis francophones!!!!

Voici le contenu de cette dernière mouture : 
- HirenBootCD 8.9, patch keyboard FR
- OCMan XPE
- UltimateBootCD 4.0.3 FR ( modifié pour incorporé Ghost/DriveImage & Partition Magic 8)


J'ai dans la mesure du possible intégré le plus possible d'outils en français pour les neophytes, mais je n'est pas eu le courage de traduire l'Hirenboot CD , dont les mises a jours sont trop fréquentes pour qu'une traduction ait une quelconque utilité, deplus il est réservé aux plus pros d'entre vous qui trouveront peut etre dans HirenbootCD des outils intéressants.

	(Remerciement à son auteur pour le formidable travail :-) )

--Infos sur l'UltimateBoot CD 4.0.3 : http://www.ultimatebootcd.com/
	Egalement Remerciement à ses auteurs et traducteurs ( c'est pas moi qui l'est traduit 		celui là, j'ai juste rajouté Ghost 8/2003, Drive Image2002 et Partition Magic8

--Info sur l'OCMan XPE : 
	Je vous présente donc fièrement la nouvelle mouture créé de A à Z par mes soins,
	Un grand ménage a été effectué depuis la mod du miniPE-Xt, je n'est gardé que le plus
	efficace des outils dont voici une petite description ( le gif animé vous montre 
	l'arborescence du menu démarrer du XPE)


L'OCManXPE inttégre la quasi totalité des driver de stockage et Réseau (LAN), ce qui lui permettra de tourner sur toutes les PC récents !

Voici l'arborescence du menu démarrer : 

Programmes=>
	-Antivirus-Spy=>
		-Ad-AwareSE FR : charge le registre du pc hote pour un scan efficace!
		-SpybotSnD FR : on ne le présente plus
		-Antivir 6 : antivirus gratuit, mise a jour possible
		-Antivir 7 : idem mais nouvelle version, mise a jour possible
		-BitDefender 8 Free : gratuit et efficace, mise a jour possible
	
	-Bench-SysInfo=>
		-CPU-Z : toutes les infos sur votre CPU/RAM
		-PC-Wizard 2007 : Informations générale sur le systeme ( comme AIDA-Everest)
		-Stress Prime : Utilitaire de stressCPU, version Mono/Dual Core.
		-Speedfan : Monitoring de température, à lancer en parallele de StressPrime.
	
	-Connexions=> Paramètrage LAN...
	
	-Deplantage=> Windows ne boot plus, la solution est là!
		-Regedit/Runscanner : Editer le registre du PC hote !
		-ERDCOmmander : Script permettant la restauration systeme, mod.Mot de passe,etc.
		-NirSoft MyUninstaller : Désinstaller proprement des logiciel du PC hote.
		-Restauration Registre : Restaure le registre d'un point de restauration.
		-ServicePE V1.0a : Modifier les services du système hote.
		-Spercleaner 2.84 : Retire les fichiers inutile de votre DD
		-Windows Registry Recovery : opération de maintenance sur le registre hote.
	
	-Disque Dur => Partition, MBR, Clonage de DD.
		-DriveImage : DI2002 et PQI explorer pour explorer les images.
		-Ghost : Ghost32 et Ghost Explorer.
		-Checkdisk : script lançant l'utilitaire Chkdsk de microsoft.
		-MBR fix : répare le secteur de boot du DD ( utilisateur avertis!!)
		-Norton Disk Doctor : un bon outil de vérification de DD.
		-Partiton Magic 8 : qui ne connait pas???
		-Partition Table Doctor : réparation et edition de la table de partition d'un DD
		-Testdisk : le meilleur outil pour récupérer une partition corrompu en "RAW".
		-DriveImageXML : gratuit et performant, par les createur de GetDataBack 
	-Gravure => 
		-SlientNight MicroBurner FR : gratuit et si pratique!!
		-Nero 6.6.0.5 : pour les inconditionnels de NERO

	-Mot de Passe-clés=>
		-MAgical Jelly Bean KeyFinder : pour retrouver tous les S/N du PC hote.
		-Windows Password Renew 1.1BETA : pour faire sauter un mot de passe session.

	-Outils =>
		-7Zip : gratuit et si parfait :-)
		-DoubleDriver: faites un backup des drivers du PC hote avant réinstallation!
		-Fab's AutoBackup : backup complet d'un compte utilisateur!
		-Migration Wizard : un peu comme Fab's Autobackup.
		-Renamer: renommer des fichiers en lot!
		-Detection Pixel Mort : un outil pour tester les ecran LCD :-)

		-Restoration : pour récupérer les fichiers récemment effacés.
		-UndeletePlus : Idem que restoration.

	-Web=>  - UltraVNC : ça peut servir ;-)
