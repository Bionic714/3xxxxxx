Heroes of Might and Magic 3 Complete:
Contains the original Heroes of Might and Magic III,
plus the expansion Armageddon's Blade and The Shadow of Death.

This release does not need to be burned to cd.
It is complete with all music/voice/movies and works flawless from the harddrive.

Manuals from the games are included as pdf files in the game folder.
Have fun with this classic and best game in the series!

Install notes:
Extract the rar archive to a folder of your choice.
You launch the game from Heroes3.exe or create a shortcut for it.
The map editor can be launched from h3maped.exe and the campaign editor from h3ccmped.exe.
