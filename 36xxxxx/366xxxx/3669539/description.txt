:: INFO ::

This little tool allows you to play online or on LAN with
S.T.A.L.K.E.R. - Shadow of Chernobyl Multiplayer.
The program supports multiple languages by language files.
(Hungarian language included with this pack)

Version: 1.1.0.1
Release date: 2007.04.21.

The pack contains:
 - stalker_mp.exe - S.T.A.L.K.E.R. Multiplayer Tool executable
 - hungarian.lang - Hungarian language file
 - S.T.A.L.K.E.R.Multiplayer.Tool-ToMikaa.nfo - The NFO file that contains the
   installataion how-to and the changelog

Web: http://tom2oo4.uw.hu/sw/index.php

For more info, see the NFO file.
