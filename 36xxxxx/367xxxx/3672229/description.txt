RipIt4Me v1.7.1.0 + required software bundle


HowTo
=======================

1. Install DVD Decryptor
2. Optionally install DVD Shrink
3. Optionally install ImgBurn
3. run RipIt4Me.exe
4. Insert DVD to RIP and start ...
5. point RipIt4Me to the the latest FixVTS version if asked
6. see info for more information
