BG CD Manager is a easy to use cataloging CDs/DVDs software. They are stored in a database on hard drive and can be viewed at any time. The program can even catalog the contents of CAB, RAR and ZIP archives.

 You can give items in your database a description, a comment or other information, to identify them more easily. BG CD Manager can automatically create descriptions for MP3, OGG, PNG, GIF, JPG, HTML file types.

 You can search in one or more CDs using one or more keywords separated with space. Once you found a file, you can open, copy or extract it directly from within the program.

 BG CD Manager support skins so it will never become boring.
