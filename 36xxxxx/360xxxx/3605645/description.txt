*********** INSTRUCTIONS ***********
************************************

1) Ouvrir le programme "Windows XP Keygen.exe" afin de produire une clef valide pour WinXP !
Quand le programme s'ouvre dans la boite de dialogue "product family" et sélectionnez "Windows XP pro. VLK"	
Cliquez sur le bouton "Generate", un numero de serie (windows) apparait immediatement ...
Laisser ce programme ouvert.

2) Ouvrir le programme "keyfinder.exe" afin de lire votre numero de serie actuel pour votre WinXP.
Mais ce programme peut aussi changer le numero de serie !
verifier que l'onglet "Microsoft windows" est ouvert, cliquez dans le menu sur "options" puis "change Windows Key"
Entrez maintenant le numero de serie generer par le programme "Windows XP Keygen.exe" laisse ouvert
Validez en cliquant sur le bouton "Change", une fenetre pop-up apparaitra, cliquez sur OK
A ce stade fermer le programme "keyfinder.exe" et re-ouvrez le pour verifier que le changement de numero de serie ait bien ete effectue ...

3) Ouvrir le programme "wga-fix.exe", une fenetre pop-up apparaitra, cliquez sur OUI et ne vous souciez pas des ecritures en langue etrangere.

4) Ouvrir le programme "MGADiag.exe" (Outil de diagnostic de validation de version originale de Microsoft), ceci afin de controler votre statut WGA
Cliquez sur le bouton "Continue", quand l'operation est finie, Cliquez sur OK

5) Pour finir, si vous avez une version crackee du fichier "LegitControlCheck.dll", effacez le du repertoire system32 de windows ("C:\Windows\Sytem32").
Ensuite telecharger à nouveau l'outil WGA de microsoft sur www.microsoft.com !
Ca y est ! votre version est reconnue comme officielle !!!!! :)
