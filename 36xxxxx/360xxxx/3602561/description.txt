TypeItIn lets you define buttons that will type in any information you want into any application. You can use TypeItIn to fill out forms on the web, or process forms at work. It's a great tool for applications where you frequently type the same thing like creating HTML code or writing standard letters or emails.
Another great use is for entering user names and passwords. You can also launch applications or web sites with the click of a button. You can create up to 200 password protected groups, each with up to 200 buttons. TypeItIn can automatically type in the time, date, month, or day of the week too. In addition, you can also Record your own typing to automatically create buttons.
