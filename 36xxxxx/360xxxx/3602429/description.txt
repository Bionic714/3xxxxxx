The XML Flash Slideshow v3 allows you to easily create high-impact XML-powered Flash Image Galleries, all in Dreamweaver MX, MX 2004, or 8.

To create an image gallery, you simply open a Dreamweaver Wizard.  Specify images, make desired customizations, and insert the slideshow in just minutes.  Any edits can be made using the edit wizard.  The customization options will satisfy the most demanding designers, while the prebuilt-defaults make usage easy for beginners.  Support for fully dynamic image galleries make the product a coder's best friend.
