Descrição:
- Título:WinAce  v 2.65

Função do Aplicativo:descompactador de arquivos

Versão2.65 Idioma bilingual





Formato alternativo de compressão de arquivos .ACE similar ao formato ZIP com quase nenhuma diferença do padrão dominante ZIP. A instalação está em alemão, mas logo após instalar o software (ah, dá pra conseguir, é só seguir o instinto...) ele mostra a opção para passar para o inglês. 

Informações Adicionais:



Software com maturidade para trabalhar com vários formatos de compressão de arquivos! O WinACE não só descomprime ACE, ZIP, LHZ, LHA, ARJ, RAR, CAB e outros, como também tem a possibilidade de gerar arquivos com essas extensões, superando a grande maioria de seus concorrentes em recursos.



Roda em Windows 95, Windows 98, Windows NT, Windows 2000, Windows Millenium, Windows XP
