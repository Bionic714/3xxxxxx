Van Dale Woordenboek(en) op DVD

Web site: http://www.vandale.nl/

Op deze DVD staan 6 woordenboeken van Van Dale. 
Je kunt deze ook integreren in Microsoft Word!
De verschillende woordenboeken kan je dan direct vanuit MS Word aanroepen.

De volgende woordenboeken staan op deze DVD:

     * Hedendaags Nederlands 

     * Nederlands - Engels + Engels - Nederlands.

     * Nederlands - Spaans + Spaans - Nederlands.

     * Nederlands - Frans  + Frans  - Nederlands.

     * Nederlands - Duits  + Duits  - Nederlands.

Ik heb alles op één DVD gezet met een makkelijk menuutje om de verschillende woordenboeken te installeren.
Je kunt elk woordenboek ook apart op CD zetten. Elk woordenboek staat apart in een Directory op de DVD,
brand zo'n Directory apart op een CD en je hebt alleen dat woorden boek.


Veel plezier.

-----------------------------------------------------------------------------------

In English:

This DVD contains 6 (meanly Dutch) dictionaries.
You can integrate these dictionaries into Microsoft Word.
And the different dictionaries are accessible within Word.
The following dictionary?s are on this DVD.

	* Dutch Today.

	* Dutch-English + English-Dutch.

	* Dutch-French + French-Dutch.

	* Dutch-Spanish + Spanish-Dutch.

	* Dutch-German + German-Dutch.

The DVD comes with an easy to use menu for installing each dictionary. 
It is possible to use only one dictionary because every dictionary has its own directory on the DVD.
By extracting one directory on the DVD and burn this on a CD you can create a bootable CD for one dictionary.


Have fun
