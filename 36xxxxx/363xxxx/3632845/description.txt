Windows Vista Home Premium - 32bit OEM Swedish

Windows Vista Home Premium är operativsystemet för hem med avancerade datorbehov. Det bidrar till att du kan använda din bärbara eller stationära dator mer effektivt och möjliggör att du kan ta del av nya, spännande digitala underhållningsupplevelser - allt med fördelen av ökad säkerhet och tillförlitlighet.

Nytt användargränssnitt
Windows Vista Home Premium har ett nytt användargränssnitt kallat Windows Aero?, som är både effektivt och snyggt. Med det nya gränssnittet är det enklare än någonsin att hitta i operativsystemet. Det är t o m enklare att utföra flera uppgifter samtidigt, tack vare en tredimensionell, realtidsbaserad, animerad vy av alla öppna program och dokument. Dessutom bidrar Windows Vista Home Premium till att du snabbt kan hitta och organisera stora samlingar av dokument, bilder, filmer, videor och musik. Tack vare integrerad sökning för hela operativsystemet bidrar Windows Vista Home Premium till att du snabbt kan hitta exakt det du letar efter.

Förbättrad mobilitet
Med Windows Vista Home Premium är det enkelt att ta med hemdatorupplevelsen vart du än går. Windows Vista Home Premium innefattar till exempel Tablet PC-teknik som gör det möjligt att interagera med en Tablet PC-kompatibel dator med hjälp av en digital penna eller med fingertoppen i stället för att behöva använda ett tangentbord. Med en dator som är utrustad med Windows Vista Home Premium och en extra Windows SideShow?-visning kan du även få åtkomst till nyckeldata även när datorn är avstängd. Det är även enklare än någonsin att dela filer med andra datorer i hushållet och hantera den bärbara datorns inställningar så att du på ett säkrare sätt kan ansluta till din favorit-WiFi-aktiveringspunkt.

Mer underhållning
Med Windows Vista Home Premium förbättras varje aspekt av de digitala underhållningsupplevelserna, däribland att visa och dela foton, videor, TV, filmer, musik, spel och annat. Med Windows Vista Home Premium kan du till exempel skapa dina egna DVD-produktioner och redigera dina egna högdefinitionsfilmer. Framför allt innefattar Windows Vista Home Premium alla funktioner i Windows Media Center för att förvandla datorn till ett komplett hemunderhållningscenter. Med Windows Media Center i Windows Vista får du nya möjligheter att ta del av musik, foton och DVD-filmer. Du kan också använda Windows Media Center för att spela in och titta på dina favorit-TV-program (även HDTV) och för att få tillgång till nya typer av underhållning och innehåll på Internet. Du kan även ansluta Windows Vista Home Premium till Microsoft Xbox 360 om du vill utvidga Media Center-upplevelsen till flera rum i hemmet.
