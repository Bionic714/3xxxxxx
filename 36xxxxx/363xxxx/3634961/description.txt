This programs contains 544 single games and 135 Maniadisks (more than 1200 rom and dsk games) which you can play in a very simple way. Just select the game and press the run button.

You can choose between four emulators:

    * FMSX-DOS (version 1.6b2.a)
    * FMSX-SDL (version 2.6.0.40)
    * NLMSX (version 0.48)
    * BlueMSX (version 2.0)

For more information: www.msxposse.com
