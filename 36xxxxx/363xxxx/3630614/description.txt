Windows Vista Light Edition es una modificación del sistema operativo Microsoft Windows Vista Ultimate.

Su principal característica es que su reducido tamaño logra que el producto pueda ser almacenado en un CD convencional. Para ello han sido extraídos numerosos componentes del mismo, la mayoría de ellos considerados obsoletos.

Windows Vista LE también apunta a sacar el máximo partido de su PC, ahorrando así en la compra de nuevo hardware.

El equipo de Windows Vista Light Edition informa que Windows Vista es una marca registrada de Microsoft Corporation. Las diversas marcas mencionadas en este sitio Web son propiedad de sus respectivos autores.
