This a self-made Utility Disc for your chipped, or soft modded Playstation 2 console.

WARNING!!!
-----------------------------------------------------------------------------
Don't download this, if you don't have either a mod-chip or some other method
of running backup/homebrew software. Also... there are a lot of readme-s and
documentation missing. Don't blame me... I've created this dvd for myself.
-----------------------------------------------------------------------------

The disc contains a lot of useful things, for booth Linux and Windows. It's
a simple collection of tools I use, as a PS2 user. The PS2 ELF-s are
also awailable through the Crystal Chip Boot Manger (I created the necesary
folders on the DVD).
The DVD also contains some other useful CD/DVD images (Uncompressed).

Most programs are (AFAIK) the newest versions available (2007.03.02).

About seeding: I have a 10k outgoing connection only, so be patient. I will seed 24/7 for ca. 1 week.
