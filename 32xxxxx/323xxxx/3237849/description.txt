Acme CADConverter is a format conversion
software for batch and vector files.

It can conveniently convert DXF,DWF and DWG files into
BMP, GIF,JPEG,PCX, DXF, DWG, SVG,PDF,
HPGL(PLT,HGL),PDF etc., and also enable the
conversion between DXF and DWG file versions
(AutoCAD R2.5-R2005). Support printing.
