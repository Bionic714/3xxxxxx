Total Textures v8 'Vehicles' is a collection of textures for applying to car models and scenes. The textures range from tyre bump maps to cool flame decals. In the case of textures such as decals, we have included the alpha-mapped TIFF files. Giving transparency to the outer edges of a map.

For The first time on a Total Texture CD we have included .dxf meshes of some of the more 'common' car objects. These include Alloy Wheels, brake calipers, dials etc. 
Contents Breakdown 



203 Full Resolution Colour Texture Maps, of which 156 have matching alpha maps.

203 Matching Bump & Alpha Maps
45 Models (both .3ds & .dxf files)
15 Paint Shaders for Lightwave, 14 for Cinema 4D and 14 for 3DStudio Max.
14 Independent Bump maps for headlamp lenses and tyres
Project Overview Tutorials,
To accompany this CD there is the added bonus of 7 Project Overviews and Tutorials that have been created by selective 3DTotal gallery artists using these textures alone.

HTML interface providing easy browsing and map selection to use with all 3D Software packages (Will work with all web browsers and is PC & MAC Compatible)
