        Install Notes (Please follow these instructions IN ORDER)

     1. Unpack the release with WinRAR or equivalent
 
     2. Close STEAM if it is running.

     3. Run patch.exe and install to ValveSteamSteamApps and wait
        until it is finished. It will take a good 10 minutes or so.

     4. Run hl2install.exe from the HL2Installer directory.
 
     5. Run cssInstaller.exe from the CSSInstaller directory.

     6. Run hl2dminstaller.exe from the HL2DMInstaller directory.

     7. Run Steam Online.reg if you want to play online.

        Edit the steamemu.ini in 
     
         ValveSteamSteamAppsUserCounter-Strike Source
        
        with your STEAM username and password and SteamOnline=1.


        Also edit the steamemu.ini in
         
         ValveSteamSteamAppsUserHalf-Life 2 Deathmatch

        with the same information if you want to play HL2 Deathmatch.

        *NOTE*
        
         You CANNOT use your previous .ini files for this update!
         You MUST edit the new ones.
 
     8. Make sure STEAM is running if you plan to play on legit servers.

     9. Run the appropriate shortcut for the game you want to play from
        your desktop.
 
        Enjoy!
