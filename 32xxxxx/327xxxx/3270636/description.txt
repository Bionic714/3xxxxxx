- ----- - Spel beskrivning - ------------------------------------- - 

Drömmen kan bli verklighet. Snart har du en nöjespark som får 
Gröna Lund att blekna. 

I Rollercoaster Tycoon 3 är din uppgift som parkchef att skapa 
en attraktiv nöjespark med glada besökare. För att klara det 
räcker det inte att bygga ett par åkattraktioner. För att allt 
ska fungera måste du se till att det finns toaletter, mat, 
sjukstuga och mycket mera. Du kan bygga parken i några olika 
teman, däribland western och science fiction. 

I spelets karriärläge gäller det att parken blir attraktiv så 
att pengarna rullar in. Du får hela tiden olika delmål som 
måste uppfyllas. Det kan vara att få ekonomin att gå med plus, 
att bygga en riktigt tilldragande åkattraktion eller att locka 
ett visst antal besökare. 

Det finns dock ett spelläge helt utan krav som jag tror kan 
bli populärt. Det heter sandlådeläge och här finns inga som 
helst begränsningar. Här kan du experimentera precis som du 
vill utan att riskera att några mål spräcks.
