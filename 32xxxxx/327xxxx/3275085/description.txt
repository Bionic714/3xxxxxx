  <h1>Arctic Torrent</h1>

  <p>Arctic is a minimal BitTorrent client. It wont have all the pretty features that other Torrent
  apps have, but focuses on low memory and cpu usage. Because it was written in C++, you dont get
  the high memory requirements of Java or high CPU usage of Python.</p>
