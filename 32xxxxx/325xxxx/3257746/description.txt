       RELEASE NOTES:

       Half Life 2, properly one of the most PROPER, proper game releases.
       Thru we have had quite a few propers, then only  minior tweaks have
       been  fixed. This is  probaberly not the last fix you will see, but 
       this fix fixes 2 important issues people are having.

       AI_DISABLE and NODE GRAPH REBUILDING bugs is a constant pain whole
       way thru the game. This fix, fixes those 2 problems!!

       INSTALL NOTES:

       Open the included file and you will noticed 2 rar files. hl2eufix 
       and hl2usfix. If you're located in EU you should use hl2eufix and
       if you're locted in US use hl2usfix. Both patches do the same, but
       due to different keyboards setups, you can't access the console on
       a EU keyboard. Now nobody should have anymore problems?

       Of course the cheats is made available, inclusive all the levels
       which you can select from when you begin a new game!!
