Innehåll i Service Pack 2 Tillbaka till början 

--------------------------------------------------------------------------------

Microsoft arbetar kontinuerligt för att förbättra företagets programvara genom att ta fram och ge ut nya uppdateringar. Med jämna mellanrum kombineras dessa uppdateringar i ett enda paket, kallat ett Service Pack.

SP2 innehåller den senaste samlingen uppdateringar för Windows XP. Dessa uppdateringar omfattar bland annat följande:

Säkerhetscenter, där du kan visa och hantera alla datorns säkerhetsinställningar på samma plats. 
Windows-brandväggen, som installeras och aktiveras som standard. Brandväggen skyddar datorn mot virus och 
andra säkerhetshot, t.ex. obehöriga användare som kan försöka få åtkomst till datorn via Internet eller ett nätverk. 
Säker filhämtning för Internet Explorer, som varnar dig för potentiellt skadliga filer och som ger dig möjlighet att 
blockera oönskade program. 
Bilagehanteraren, som skyddar Outlook Express, Windows Messenger och Internet Explorer mot virus som 
kan spridas på Internet. 
Blockering av popup-fönster i Internet Explorer, som gör det säkrare och roligare att surfa på Internet 
eftersom oönskade popup-fönster blockeras. 
Windows Movie Maker 2.1, som installeras när du installerar SP2. 
Den här versionen av Windows Movie Maker innehåller uppdateringar för Grupprincip, som kan 
vara intressanta för systemadministratörer. Versionen innehåller också uppdateringar som hjälper dig att 
öka säkerheten på datorn. Mer information finns på webbplatsen Windows Movie Maker.
