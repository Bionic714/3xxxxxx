Stomp Simple Backup, Version 4.85

VERITAS Software Corp. 

Stomp Simple Backup is an easy-to-use program for backing up
your important files and restoring them if something ever
goes wrong with your computer. To use Stomp Simple Backup,
simply open the program from the Windows Start menu and follow
the on-screen instructions. 

Boot from CD Disaster Recovery on FAT32 or DBCS systems.
