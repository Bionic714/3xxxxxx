Adobe Atmosphere v.1.0

Adobe Atmosphere is a professional authoring tool for assembling and
creating 3D interactive stage sets. It gives the web designer the ability to
present a rich variety of interactive content
                                                                     Features:                                                     
  - High-definition 3D Objects                                     
  - Web page integration                                              
  - Real-time Physics ( Havok )                                       
  - High-quality Environments
