  Game Notes: 

  Virtual Pool Hall has six new pool games, Snooker, and two billiard games. 
  Virtual Pool Hall contains all elements of Virtual Pool 2 with significant
  enhancements. Listed below are new features added for Virtual Pool Hall.
  Those unfamiliar with VP2 can learn more here : 
  http://www.celeris.com/games/pool/index.html
  

  FEATURES :
  
  New Tables

  Virtual Pool Hall has three new tables as well as the championship size 
  9x4.5 foot pool table from VP2.
  
  Players can visit the Bar and play pool on a 7x3 foot bar table. The smaller 
  table is a fun alternative to the championship table and is common in bars 
  across the USA.
  
  Snooker can be played on a regulation size 6x12 foot snooker table. Players
  that have never been on a 6x12 Snooker table are in for a surprise.
  Three Cushion Billiards is played on a 5x10 foot table.
  
  Each table is in a separate pool hall.
 
  Simulation

  The simulation has been enhanced to allow Snooker and pool games to be 
  played on any of the tables except the Billards table.

  The addition of Billiards showed many shortcomings to the VP2 Masse model.
  Virtual Pool Hall has a new Masse model that fixes errors including squirt
  and spin computations.

  The rail response has been adjusted for less spin reduction on certain 
  types of shots. 

  Maximum shot velocity has been decreased for shots other than breaking.

  Table parameter pre-sets for Amateur, Pro, and Championship tables have been
  adjusted, based upon user feedback and additional real life testing, 
  to the following:

  Amateur - slower rails
  
  Pro - slower table, tougher pockets
 
  Championship - slower table, tougher pockets
