Instrucciones: El ZXBInstaller tiene la información de la configuración de FTP de los Dash y del mismo ZXBInstaller y otras cosas más en la Sección "1) Información". Todos los Dash que se instalen en C/ lo harán como evoxdash.xbe, si tienes una BIOS antigua que busca como Dash sólo al xboxdash.xbe, puedes usar el Explorador de Archivos para modificar lo que necesites. 

El ZXBInstaller tiene la habilidad de instalar el Exploit de las fonts usando el método de Hot Swap, les recuerdo que antes de instalar el Exploit se debe formatear por completo C/ y luego instalar el Exploit. Recuerden revisar si su Kernel admite este Exploit. 

Las aplicaciones, emuladores, juegos homebrew y Dashboards al instalarlos en E/, F/ o G/ se crearán las siguientes carpetas: 

Aplicaciones -> Apps 
Emuladores -> Emulators 
Juegos Homebrew -> Games 
Dashboards -> Dashboards.
