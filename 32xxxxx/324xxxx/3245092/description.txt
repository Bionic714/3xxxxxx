Install Note:

You must Install first the full game then you must start the patch.


Setting Preferences :
 
     If you wish to override these defaut seetings, you may do so by adding
     the folling letters (separated by spaces) after typing monkey2 when 
     when you start the game.

     for example, to start the game in MCGA gfx with Adlib Soft type:
    
     monkey2 m a 
    
     a = Adlib Music Syntheizer or Soundmaster II Sounds
     s = SoundBlaster sounds
     r = Roland sounds
     i = Normal sounds (internal speaker)
     m = MCGA gfx mode
     v = VGA gfx mode
     mo = Use mouse control
     j = use joystick control
     k = use keyboard control
