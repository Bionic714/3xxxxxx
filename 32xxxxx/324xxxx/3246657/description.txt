This is Internet Explorer plug-in that helps you visit web pages in foreign languages. It can translate between 13 languages:<br />
 English, German, French, Italian, Spanish, Portuguese, Dutch, Russian, Greek, Chinese (simplified), Chinese (traditional), Japanese, Korean.<br /> <br />
Using this handy tool is extremely easy. Just navigate your browser to desired web page, select source and destination language and press Translate button. In a few seconds browser will load translated page that looks just like the original page. The only difference is text translated to desired language - links still work and images are on the right place.
