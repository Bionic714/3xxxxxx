AnyDVD är en enhet, som avkodar DVD-Filmer automatiskt i bakgrunden. DVD´n synes vara oskyddad och regionsfri för alla program och likaså för Windows operativsystem. Med AnyDVD's hjälp kan kopieringsverktyg som CloneDVD, Pinnacle Instant Copy, InterVideo DVD-Copy, etc. kopiera CSS skyddade filmer. Med hjälp av AnyDVD kan du titta på filmer med icke matchande regionskoder med vilket DVD Spelarprogram du önskar! AnyDVD avkodar inte bara DVDs: AnyDVD tillåter dig också att spela, kopiera och rippa skyddade Ljud CDs!
