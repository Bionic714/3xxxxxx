IP*Works! Secure Tunnel adds SSL functionality to any server. With    º

   º  secure tunnel you can upgrade your existing server to a secure        º

   º  server with minimal effort and time. There is no need to go through   º

   º  the painstaking and costly process of setting up individual secure    º

   º  servers.                                                              º

   º                                                                        º

   º  Secure Tunnel is not limited to the World Wide Web. Secure Tunnel     º

   º  can make any type of server secure. For email security you can        º

   º  configure Secure Tunnel to enable a secure SMTP or POP server.        º

   º  Secure Tunnel works with FTP to create secure connections for file    º

   º  transfer. With Secure Tunnel you can even add SSL functionality to    º
