Jogue no novo modo para Um Jogador com uma missão de 30 fases e modo Deathmatch. Mais cinco novos Modos de Treinamento incluindo Corrida de Ovelhas (você tem dois minutos para pegar o máximo que puder), Caixas Malucas (use a corda para pegar as caixas), Eutanásia (pegue a Velhinha), Tiro ao Peru e Tiro ao Alvo.
Mega-diversão no modo Multiplayer que agora inclui Handicap (adicione ou reduza a energia de uma equipe), Equipes Aliadas (junte-se com outras pessoas para acertar outro jogador), Cartas de Doação (uma chance para continuar com o que for melhor para você), Doença de Vermes e Corrida de Pontos.
O Fim de Tudo Como Nós Conhecemos!
----------------------------------------------------------------------
Configuração Mínima
Pentium 100 MHz (Pentium 150 MHz recomendado)
Windows 95/98
32 MB de RAM
Unidade de CD-ROM de 2X
50 MB de espaço livre no disco rígido
Placa SVGA de 1 MB (4MB recomendados)
Placa de som de 16 bits compatível com Windows 95/98 (incluso patch para jogar no Windows XP)
DirectX 6.0 (incluso no CD do jogo)
