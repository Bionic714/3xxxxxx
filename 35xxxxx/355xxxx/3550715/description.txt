Johdanto:

OEM-laitevalmistajat, jotka kokoavat tietokoneita laiteosista, voivat tämän paketin työkaluilla esiasentaa ja määrittää asetukset seuraaville
Windows XP- ja Windows Server 2003 -tuoteperheen tuotteille:

Windows XP Home Edition ja Service Pack 2 (SP2) 
Windows XP Professional ja Service Pack 2 (SP2) 
Windows Server 2003 Standard Edition ja Service Pack 1 (SP1) 
Windows Server 2003 Enterprise Edition ja Service Pack 1 (SP1) 
Windows Server 2003 Datacenter Edition ja Service Pack 1 (SP1) 
Windows Server 2003 Web Edition ja Service Pack 1 (SP1) 
Windows Server 2003 Stardard Edition for 64-bit Extended Systems (saatavana vain englannin- ja japaninkielisenä) 
Windows Server 2003 Enterprise Edition for 64-bit Extended Systems (saatavana vain englannin- ja japaninkielisenä) 
Windows Server 2003 Datacenter Edition for 64-bit Extended Systems (saatavana vain englannin- ja japaninkielisenä) 
Windows XP 64-Bit Edition Version 2003 for 64-bit Extended Systems (saatavana vain englannin- ja japaninkielisenä) 
Windows XP 64-Bit Edition Version 2003 for Itanium-based systems (saatavana vain englannin- ja japaninkielisenä) 
Käytettävissä olevat Windows-käyttöjärjestelmät määräytyvät OEM-käyttöoikeussopimuksen lajin ja kielen mukaan.
varmistaa, että Windows-käyttöjärjestelmän perustoiminnot näkyvät kaikille käyttäjille samanlaisina 
mahdollistaa kaikkien Windows-tuotteiden mukauttaminen ja varustaminen laitevalmistajan tuotemerkillä 
mahdollistaa käyttöjärjestelmään sisältymättömien lisäsovellusten ja laiteohjaimien esiasentaminen 
vähentää ohjelmien laitonta käyttöä 
tarjota joustavat menetelmät ja työkalut, joiden avulla laitevalmistaja voi optimoida tuotantoprosessinsa riippumatta siitä,
valmistuuko kuukaudessa 10 vai 10 000 tietokonetta.
