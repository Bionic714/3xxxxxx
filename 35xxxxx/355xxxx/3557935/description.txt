      Description:
	Many users have had an "Evaluation" watermark on their
	desktop after installing Windows Vista Final. Use this
	to fix it up.

      System Requirements:  
	Windows Vista v6.0 (Build 6000)


      Install Notes:
	(1) Extract "user32.dll.mui" to your desktop.
	(2) Reboot into safe-mode (F8 on boot).
	(3) Go to: "C:\Windows\System32\en-US\".
	(4) Backup your "user32.dll.mui" file (rename to "user32.dll.mui_backup").
	(5) Copy the "user32.dll.mui" from "Desktop" to "C:\Windows\System32\en-US\".
	(6) Reboot.
