This is the latest Tomtom and scandinavian maps. Only the necessary files ;)

Copy all to your SD-Card
The install should start automaticly, if not run the cabs in 2577 dir.
When asked to enter code for the maps cancel the install.
Copy TomTom Navigator.exe from CRACK dir to your install dir.
Start Tomtom again and enjoy this sweet version of the app :)

After all is installed you can delete the Crack & 2577 dir, but Guam is needed when changing maps.


Tested on WM2003SE & WM 5 Pocketloox T830
