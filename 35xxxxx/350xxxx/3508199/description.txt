Gör så här för att installera i Windows XP. Det går inte att köra installations-CD:n rakt upp och ner för de drivrutinerna fungerar inte i Windows XP.

1. Mounta eller bränn imagen.
2. Kör webinstaller-filen som ligger med.
3. Välj att du vill installera programmen från installations-CD:n.
4. Följ resten av instruktionerna.
