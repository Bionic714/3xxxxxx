Nettiasennus
------------

Note: This program is originally designed to Finnish users and Windows XP SP2 Finnish OS but now it works in ENG/FIN: XP,MCE2005 & W2K3. (W2K - half working...) Details, homepage and also program itself language is Finnish.

Kotisivu: http://www.jyrkihuovinen.net/nettiasennus/nettiasennus.htm
Versio: 1.05 pvm: 26.9 2006
Koko: 230kt

Nettiasennus -ohjelma lataa ja asentaa internetistä automaattisesti käyttäjän valitsemat ilmaisohjelmat. Nettiasennus on tarkoitettu ensisijaisesti käytettäväksi Windows XP Home tai Pro SP2 Finnish -käyttöjärjestelmässä joka on ns. "puhdasasennus". Suurin osa ohjelmien asennuksista toimii kuitenkin silti jos vain normaaliin tapaan poistaa koneeltaan ensin aikaisemman version kyseisestä ohjelmasta. Suurin osa ohjelmista asennetaan oletusasetuksin muutamia poikkeuksia lukuun ottamatta. Esim. Daemon Toolssin mukana ei asenneta mainoskompononenttia. Daemon Toolssin asennus käynnistää tietokoneen uudelleen automaattisesti asennuksen loppuvaiheessa ja jatkaa asennusta automaattisesti seuraavan kirjautumisen yhteydessä. Nettiasennus ohjelmaan on otettu mukaan vain niitä ilmaisohjelmia jotka eivät pakotetusti asenna mainoskomponenttia tai muuta vastaavaa haittaohjelmaa.

Nettiasennusta ei tarvitse erikseen asentaa koneelle.

Nettiasennus ei jää koneeseen pyörimään taustalle paitsi jos asennat Daemon Tools ? ohjelman. Daemon Toolssin asennuksesta johtuen Windows\system32 ? kansioon jää asennuksen jälkeen Jalkiasennus.exe. Tiedosto ei jää silti mitenkään jatkossa pyörimään ja sen voi poistaa halutessaan Daemon Toolssin asennuksen jälkeen.

Nettiasennus on ns. freewarea eli vapaasti käytettävissä ilmaiseksi. Käytät Nettiasennus ohjelmaa täysin omalla vastuullasi. Ohjelmien lataus tapahtuu tällä hetkellä 3 eri palvelimelta (Funet, Saunalahti ja Vuokrapalvelin.net). Nettiasennus päivittää itsensä automaattisesti jos uusi versio saatavilla netistä. Ohjelmaa päivitetään tällä hetkellä n. kerran kuukaudessa.

Nettiasennuksen laitevaatimukset jne.:

    * Suomen- tai englanninkielinen Windows XP Home tai Pro jossa SP2 asennettu. Toimii myös Windows 2003 Server SP1:ssä sekä MCE 2005:ssä. Huom. jos käyttöjärjestelmä on englanninkielinen pitää maa- ja näppäimistöasetukset käyttöjärjestelmästä olla asetettuna suomalaisten asetusten mukaisesti.
    * Internetyhteys täytyy olla toiminnassa (laajakaista)
    * Windows asemassa väh. 1,5 Gt vapaata kovalevytilaa
    * Sinun täytyy olla kirjautuneena Windowsiin käyttäjätunnuksella jolla on Järjestelmänvalvojan oikeudet käytössä.
    * Nettiasennus on testattu ja optimoitu 1,5Ghz tai nopeampiin tietokoneisiin.
    * Kaikkien koneen komponenttien ajurit on hyvä olla valmiiksi asennettu toimimisen varmistamiseksi. Esim. jos Äänikortin ajuria ei ole asennettu ? Winamp asennus jää kesken.
    * Käytössä ei saa olla palomuuria joka kysyy lupaa internetyhteyden käytölle. Esim. Kerio tai F-Secure Internet Securityn kanssa Nettiasennus ei toimi. Ainoa palomuuri joka on tuettu / jolla toimii on Windowssin oma palomuuri.

Tiedossa olevat ongelmat:

    * Joissakin koneissa ohjelmien lataus näyttää jäävän 1%:n kohdalle paikoilleen. Todellisuudessa kyseinen ohjelma kuitenkin silti latautuu taustalla loppuun saakka ja asentuu myös.
    * Nettiasennuksen käyttöliittymä ei näy / toimi kunnolla jos Windowsissasi on käytössä ?Suuret fontit?.
