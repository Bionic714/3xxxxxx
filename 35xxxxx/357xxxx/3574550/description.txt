INTRODUCCIÓN
El sistema Microsoft Office 2007 es un conjunto completo de software de servidor y escritorio que ayuda a optimizar el modo en que usted y su equipo hacen negocios. Esta última versión ofrece nuevas capacidades para aumentar la productividad personal, simplificar el trabajo en equipo, optimizar los procesos y la administración del contenido empresarial y mejorar el conocimiento de la actividad en la organización...

La versión Enterprise que se pueden descargar a continuación incluye...

    * Access 2007: Tome decisiones fundadas realizando un seguimiento, informando y compartiendo la información de manera eficaz...
    * Excel 2007: Aprenda cómo Excel 2007 proporciona herramientas eficaces que permiten analizar, compartir y administrar información a fin de tomar decisiones fundadas...
    * Groove 2007: Utilice Groove Server 2007 para implementar, administrar e integrar el software de Microsoft Office Groove 2007 en su organización y para ayudar a sus equipos de negocios a ganar visibilidad, perspectiva y control sobre sus carteras...
    * InfoPath 2007: Recopile información de manera eficaz y amplíe el alcance de los procesos empresariales utilizando formularios electrónicos enriquecidos...
    * OneNote 2007: Recopile toda la información en un solo lugar, encuentre lo que busca de forma rápida y trabaje con otros usuarios desde la misma página...
    * Outlook 2007: Céntrese en los asuntos más importantes utilizando Outlook 2007 para establecer prioridades, controlar y buscar información de forma instantánea...
    * PowerPoint 2007: Cree presentaciones dinámicas e impactantes de forma más rápida que nunca utilizando las nuevas capacidades gráficas en una interfaz de usuario funcional...
    * Publisher 2007: Mejore la forma en que administra los materiales y las actividades de marketing. Compruebe las herramientas que le guían paso a paso desde el concepto hasta la entrega...
    * Word 2007: Cree más rápidamente que nunca espectaculares documentos conectados utilizando herramientas de edición eficaces presentadas en una interfaz de usuario funcional...


REQUERIMIENTOS
Para actualizar de Microsoft Office 2003 a la versión 2007 no debería ser necesario actualizar su hardware, aunque puede que necesite instalar un sistema operativo compatible. Al actualizar de Microsoft Office 2000 u Office XP a la versión 2007 tendrá que asegurarse de que el hardware y el sistema operativo cumplen los requisitos de sistema mínimos para Microsoft Office 2007...

Se recomienda realizar una evaluación del equipo antes de instalar los productos del sistema Microsoft Office 2007 para comprobar que el equipo cumple o supera los requisitos de sistema mínimos...

Los programas del sistema Microsoft Office 2007 son compatibles con Windows Vista Beta 2, pero no con las versiones anteriores de Windows Vista. El cliente de programas del sistema Microsoft Office 2007 es una aplicación de 32 bits y se ejecuta en una plataforma Windows de 64 bits (Windows XP y Windows Server 2003) a través de WOW64...
