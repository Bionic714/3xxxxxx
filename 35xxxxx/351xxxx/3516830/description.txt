Calendar Mine is the only calendar you will ever need, providing accurate date information starting in 1583 up until the year 2500. It is a theme based, multi-cultural, personal desktop calendar. We have taken the approach that a personal desktop calendar should bring the beauty of a wall calendar and the availability of a desktop blotter to your PC.

Setup instructions:-

1. Run calmin20

2. Copy Calendar Mine 2.0 Build 392_crk to calendar mine directory and run patch

3. Run Islamic theme if you like

4. Enjoy!
