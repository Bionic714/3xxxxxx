Les Cd ont été :
_ dump depuis mes cd originaux
_ convertis de Pal vers NTSC
_ image recentrée
_ convertis grâce Ã  l'utilitaire popstation
_ possèdent une sauvegarde propre
_ possèdent une icône et fond propre

Installation :

_ le jeux PSX nécessitent le custom firmware 3.02 OE-B (minimum) par Dark_aleX
_ glisser les répertoires 'FFVII - CDX/' vers 'PSP/game/' de votre psp.

Note :

_ une bande noire de 3mm subsiste comme sur l'ensemble des jeux PAL
ce qui est probablement dû à la différence du nombre de lignes entre le PAL et le NTSC
_ aucun problème de sauvegarde entre CD : CD1 -> save -> CD2 -> save -> CD3


ATTENTION : veuillez vous assurer de posséder l'original de ce jeux (comme moi).
