This program put a icon in your System Tray. This icon show the state of your Bloq May key, Bloq Num Key and your Bloq Scroll key. It show too when you press the Ctr, Alt or Shit keys.
This program needs Microsoft Framework 2.0 to run.
This program is freeware, you can dristribute it if you like it.
Enjoy :)
