NEW! XtraTools 2007 - a set of applications consisting of more than 20 independent system utilities that are complimentary to the standard MS Windows utilities. XtraTools increases productivity and makes working with MS Windows easier.

XtraTools 2007 contain tools:

WinTools.net - a suite of tools for increasing MS Windows operating system performance. WinTools.net cleanly removes unwanted software from disk drives and dead references from the Windows registry. WinTools.net puts you in control of the Windows start up process, memory monitoring and gives you the power to customize desktop and system settings to fit your needs. 

Armor Tools - a suite of tools to ensure your privacy and keep your sensitive information secure. Armor Tools makes it easy for you to remove traces of your activities from the computer - deleting documents, programs and other files so that they can not be recovered. Armor Tools also features tools for making additional safety related adjustments including five independed tools...

Extra Drive Pro - a suite of tools for creating Extra drives on your computer (physical drives remain unchanged and are not divided into additional partitions). The program includes tools for creating Encrypted drives (he drive is encrypted by 256-bit / 32 - symbols) password using "AES256" algorithm in real-time mode), Virtual CD-ROMs, Virtual Drives based on files, Virtual Drives based on folders, RAM-drives...

RAM Saver Pro - an easy-to-use RAM optimizer tool that will keep your computer running faster. It increases the operation system performance by making more memory available for your applications. "RAM Saver Pro" will: defragment system memory for faster access time; recover memory leaks from poorly behaved applications; temporarily flush unused libraries out to disk and so on.
