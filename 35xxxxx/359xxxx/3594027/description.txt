MCTS Microsoft Dot Net FrameWork 2.0 Application Development Foundation Training Kit

This training kit is designed for developers who plan to take Microsoft Certified Technical
Specialist (MCTS) exam 70-536, as well as for developers who need to know
how to develop applications using the Microsoft .NET Framework 2.0. We assume
that before you begin using this kit you have a working knowledge of Microsoft Windows
and Microsoft Visual Basic or C#.
