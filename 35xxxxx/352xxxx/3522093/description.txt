A Live Linux DVD, set up as a screensaver for Microsoft Windows. Screenshot herehttp://linuxtracker.org/uploads/2189.png , and more information here http://linuxtracker.org/torrents-details.php?id=2189 . Download, burn to DVD, install in the usual way for Microsoft Windows applications, and configure in the usual way for Microsoft Windows screensavers. 
Fastest way back to Windows is Ctrl-Alt-F then click the red cross.

Enjoy !
