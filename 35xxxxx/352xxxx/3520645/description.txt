Frets On Fire, Guitar Hero Music Addon

48 Songs, imported from PS2 Guitar Hero DVD with FoF built-in importer
You need the game first, if you don't, go http://louhi.kempele.fi/~skyostil/uv/fretsonfire/

Extract all files to FretsOnFire\data\songs\ folder. That's it.
