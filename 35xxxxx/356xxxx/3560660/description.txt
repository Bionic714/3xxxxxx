La guia Campsa interactiva en CD-ROM. La original.

No seriales ni cracks. Simplemente extrae el archivo zip y obtendras la imagen ISO del disco.

Si usas el Nero para grabar tus discos elige "Copiar y copia de seguridad">"Grabar imagen en disco" desde el menu del Nero 

StartSmart. Como archivo de imagen de disco selecciona la "guia campsa 2006.iso" que has descomprimido antes. Sigue las 

instrucciones del Nero o consulta su ayuda.

Una vez has grabado el CD ya puedes instalar normalmente, pero ten en cuenta lo siguiente:

Si no tienes el .NET 1.1 de Microsoft instalado el instalador de la Guia Campsa te lo instala automaticamente. No se necesita 

conexion a Internet para esto. Incluso si tienes el .Net 2.0 ya instalado igualmente se te instalara la version 1.1.  No es 

posible cambiar esto. No hay problema, la version 2.0 del .NET framework se conserva.

Tambien puedes instalar la Guia en un llavero de memoria USB. Necesitas 256 Megas.

Para ello haz lo siguiente.

Abre el CD del la Guia.

Abre la carpeta "Backup"

Veras que hay una carpeta llamada "Guia Campsa interactiva 2006 en Cd Rom"

Seleccionala, copia y pega en tu llavero USB.

Explora el llavero (una vez copiado) y entra en la carpeta "X:\Guia Campsa interactiva 2006 en Cd Rom\binback\" 

Veras un fichero ejecutable "GCI2006.exe".

Si haces doble clic en el se ejecuta la Guia Campsa.

Para facilitar el acceso puedes crear un enlace en la carpeta raiz del llavero.

Clic con el boton secundario del raton>Nuevo>Acceso directo>[Selecciona como destino el fichero arriba mencionado]
