                                 Release Notes
-------------------------------------------------------------------------------
The Small Edition stock icon sets will give an instant face-lift to your 
application, adding color and a more appealing visual style. All of the sets 
have been designed to work together seamlessly, so no matter which icons you 
require, you know they will work perfectly. Small Edition icons are ideal for 
toolbars, TreeViews, listboxes, combo controls, web sites, product documentation 
- virtually anywhere you need a high quality image.

* All images delivered in 16x16 and 24x24 pixel sizes 
* 24bit BMP format 
* 8bit GIF format 
* 3 image states; Normal, Hot and Disabled 

Each set includes a handy PDF format Image Guide, listing every icon and 
filename in the set. If you are an LMD Tools user, compiled imagelists for the 
LMD components are also included.

The icons are suitable for all tools, including Borland Delphi & C++ Builder, 
Microsoft Visual Studio, Dreamweaver and any other development platform.
