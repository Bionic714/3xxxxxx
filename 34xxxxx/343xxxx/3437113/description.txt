IconBuilder xp is a filter for use with Adobe Photoshop that makes creating icons a snap. By harnessing the professional power of Photoshop and Photoshop layers, IconBuilder provides icon artists with the most complete tool set possible for modern icon design.


Summary of IconBuilder xp Features:
   Creation of 32-bit icon formats for Windows XP
   Creation of traditional Windows based .ico files
   Supports transparent / translucent masking of icons
   QuickBuild creates all resources in a single click
   Extract any PC icon into Photoshop for editing
   View icons against a variety of desktop colors
   ToolTips for novice and advanced users
   IB does it all - no need for additional icon editors
