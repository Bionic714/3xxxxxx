Radio Zone  
The web site for internet radio. Here you will find information about the "SilentNight Radio" and its internet radio stations.

Introduction
The SilentNight Radio is an easy to use windows application for listening to internet radio stations from all over the world.

It has recording and playback features, and a mp3 tag editor. The radio stations are sorted in genres, a ever growing list.
You can select by categories, like Country, Hits, Classical, Rock, Norwegian, Swedish, Gospel, Comedy...etc.

Included with the radio, you find a very easy to use cd-burner, to create audio-cd's from the recorded material. Iit's possible to schedule a recording and do a so called divided recording from the selected station. The system will record and name each mp3 file with artist name and song.
