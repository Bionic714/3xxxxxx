Microsoft Visual Studio Tools for the Microsoft  Office  System  is  a  new
  technology that brings the power and productivity of Visual Studio .NET and
  the Microsoft .NET Framework  to  business  solutions  built  on  the  next
  versions of Microsoft  Word  and  Microsoft  Excel. With  this t echnology,
  developers using Visual Studio .NET 2003 can  use  Microsoft  Visual  Basic
  .NET and Microsoft  Visual C# .NET to write code  behind  Microsoft  Office
  Word- and Microsoft Office Excel-based applications.
