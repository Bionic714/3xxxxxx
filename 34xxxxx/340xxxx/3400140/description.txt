Create Web menus without coding. No DHTML or JavaScript knowledge is required.

JavaScript export - save your Web menu in a simple to use .js file, ready to add to your Web page
menu styles, multiple menu styles (for example "file cabinet" style)
save and open, save the menu to a file and open it later for editing

fast page loading - generated JavaScript is light, reliable and size optimized
