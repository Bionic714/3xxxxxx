#develop (short for SharpDevelop) is a free IDE for C# and VB.NET projects on Microsoft's .NET platform. It is open-source (GPL).

This file is a setup version for SharpDevelop 2 on .NET 2.0.

Note this is a non-stable release, the latest stable release of SharpDevelop is avaliable at the developer site: http://www.icsharpcode.net/OpenSource/SD/Download/
