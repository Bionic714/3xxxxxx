Windows Unattended CD Creator is a very small application that will help you create an user-customized Windows bootable CD. You can choose which drivers, components or hotfixes to be installed and you can even set your network settings. All you need to do in a normal Windows installation is taken cared of by this little application, so you can install your operating system in the fastest way possible.
