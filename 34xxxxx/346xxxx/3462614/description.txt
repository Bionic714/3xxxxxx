GamePadz is built to serve the needs of hardcore gamers with large game collections. It currently supports over 750 Xbox games and the list grows all the time via automatic updates. The website at GamePadz.com offers discussion forums, and eventually a wide range of various content ranging from reviews, news, cheats, trainers, and more.

* exhaustive database of almost every Xbox game ever released in the USA
* painlessly add and remove titles from your "My Games" list
* sort "All Games" and "My Games" list by any of over 28 types of information
* game descriptions and instructions
* rotating 3D box art viewer
* automatic game data and media updates
* multiple language localization support
* save "My Notes" for any individual game
