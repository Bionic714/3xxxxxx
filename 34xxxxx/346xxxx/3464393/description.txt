Real_Videoteque is a simple program to manage own videoteque (in special way the DVD-Rip films masterized on DVDs).

Supported Languages:
- Italian
- English
- German
- Spanish
- French

Supported Resolution
- 800x600
-1024x768
-1280x1024

Functions:
- Inserimento Films/DVDs
- Modify Films/DVDs
- Cancel Films/DVDs
- Search Films/DVDs
- Printable Version (it creates an HTML file with the selected films)

Some advice:
To modify or to cancel faster a film I suggest to double-click on the film chosen to open immediately the "Modify Film" windows,  the same if yuo want to delete a film (in this case you have to press the DEL botton)
