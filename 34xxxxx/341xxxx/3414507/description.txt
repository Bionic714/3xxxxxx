Loader for Flight One msfs2004 addons.

Additionally you need:
1. .key file for a product (for example, http://thepiratebay.org/details.php?id=3238460 for Ultimate Traffic)
2. original .exe file downloaded from www.flight1.com (with red plane as an icon)

How to use:
1. Run the loader
2. Choose the .exe file from point 2
3. Click `Click here to reinstall'
4. Choose the .key file from point 1
5. If credit card number is requested - type any

See files in the archive for further info.
