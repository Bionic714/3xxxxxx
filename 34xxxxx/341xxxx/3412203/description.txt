==================================================================
Configuration requise
==================================================================

Configuration minimale :

Système d'exploitation - Mac OS 9.1 ou ultérieur (natif OS X)
Processeur - processeur G3 à 350 MHz
Mémoire - 128 Mo de RAM
Carte vidéo - ATI Rage 128 ou mieux (carte 3D disposant de 16 Mo de mémoire vidéo ou plus)
Espace disque - 600 Mo libre sur le disque dur
Vitesse du lecteur CD-ROM - 4x

Configuration recommandée :

Système d'exploitation - Mac OS 9.1 ou ultérieur (natif OS X)
Processeur - processeur G3/G4 à 500 MHz
Mémoire - 256 Mo de RAM
Carte vidéo - GeForce 2MX, Radeon ou mieux
Espace disque - 600 Mo libre sur le disque dur
Vitesse du lecteur CD-ROM - 8x

==================================================================
Amélioration des performances du jeu
==================================================================

Certaines machines profiteront de ces réglages des options vidéos/sonores :

	- Essayez d'abaisser le détail des textures dans le menu 'Options'. C'est notamment utile pour les ordinateurs disposant de seulement 128 Mo de RAM.

	- Si la résolution vidéo est plus élevée que 640x480, essayez de la réduire à 640x480 dans le menu 'Options'.

	- Si la profondeur des couleurs est sur 32-bits, essayez de la réduire à 16-bits à partir du menu vidéo.


==================================================================
Autres conseils
==================================================================
Reconfiguration des touches :

Si vous reconfigurez les touches, vous devrez appuyer sur ESC pour quitter la sélection.

Si vous voulez revenir aux réglages par défaut, vous devrez supprimer le fichier Game.INI situé dans votre dossier Harry Potter II. Cela rétablira tous les réglages des touches à leurs valeurs par défaut.

Il y certaines combinaisons de touches qui peuvent être reconfigurées soit dans Quidditch, soit dans le jeu normal. Si vous reconfigurez ces touches, le jeu fonctionnera normalement, mais vous pourriez le trouver confus. Si vous voulez revenir aux réglages par défaut, suivez les instructions ci-dessus sur la suppression du fichier Game.ini.
