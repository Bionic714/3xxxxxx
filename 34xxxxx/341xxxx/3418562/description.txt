The PC game Black & White 2. These files are cue and ccd files. So you'll have to use Daemon Tools(4.00) to make this work.

You play as a god to help your people(greeks). You can be either good or evil. You have a creature who also can be good or evil. They will explain more in the game.

Language: English

Requirements:
Operative system - Windows XP/2000
Processor Speed - 1600 MHz(1.6 GHz)
System memory - 512 MB
DirectX version - 9.0
Display Device - probably at least NVIDIA GeForce - FX 5700
Display memory - 64 MB

First, set the number of devices to 4. Then mount all devices like this:

Device 1[F:] :black.and.white.2.disc.1.cue
Device 2[G:] :black.and.white.2.disc.2.cue
Device 3[H:] :black.and.white.2.disc.3.cue
Device 4[I:] :black.and.white.2.disc.4.cue

After you've mounted all devices, just click the Install button and then choose which directory you wish to install this. When you're being asked about the CD-KEY, just start keygen.exe and write the CD-KEY in the installation boxes. During the installation, they will also tell you to insert Disc 2, Disc 3 etc. When they are asking you this, just click Browse and double-click on the Device you've mounted Disc 2 on. Do the same thing when they're telling you to insert Disc 3 and 4. When the installation is complete, you should be able to play this game without any problems if you leave the mounted devices as it is. If you unmount them or exit Daemon Tools, the game will not work. But you can always mount the devices again and play the game without installing the game one more time =)

Enjoy!!
