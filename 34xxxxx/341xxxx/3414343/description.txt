Collection of 8 themes for Windows XP/2003 Server, downloaded from www.themexp.org before they started wrapping all files into *.exe archives. This means you don't have to use StyleXP, but patch uxtheme.dll and install all themes manually ;] For more info on that, google it! DLL and instructions are not included, only themes.

Themes:
* DosX Aqua
* FF-XBox
* Forever Blue
* Lamborgini
* Senitel
* Temujin
* TRT Tech
* Windows Recall
