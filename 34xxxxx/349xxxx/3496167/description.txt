This is a batch file auto installer. 
It will install H264 / X264, FFDShow & Winamp. 
Everything goes in all the right places. 
Once installed you are ready to watch H264 streams and movies. 

Note this is configered for H264, not necessarily all video file types. 

To edit settings for other formats, choose. 
1. START 
2. FFDSHow 
3. VFW Codec Configuration. 
4. Choose "Decoder" from the two options on the top left. 
5 Right click the additional file type you want to set it for.
