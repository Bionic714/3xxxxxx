      In-game Hotkeys:

         NUMPAD 1: Infinite Health
         NUMPAD 2: Infinite Ammo/Grenades/etc
         NUMPAD 3: Infinite Oxygen
         NUMPAD 4: Infinite Medkits
         NUMPAD 5: Disable Alarm Meter
         NUMPAD 6: Super Speed
         NUMPAD 7: Super Zoom              (see notes)
                Z: Zoom In
                X: Zoom Out
         NUMPAD 8: One Hit Kill
         NUMPAD 9: Freeze AI               (see notes)
               F1: Fly                     (see notes)
               F2: Wireframe Mode          (see notes)
         NUMPAD 0: Toggle In-game Menu     (see notes)

      IMPORTANT NOTES:

      - For the INGAME MENU and WIREFRAME MODE option to work, you have to 
      run the trainer BEFORE the game and KEEP IT RUNNING WHILE THE GAME IS 
      STILL RUNNING. Wireframe mode allows you to see through walls and 
      obstacles to locate enemies.

      - Most of the options can be toggled ON and OFF by pushing the hot key 
      again, you will hear a confirmation beep sound.
      
      - To use Fly, first enable the option with assigned hotkey, then press 
      Crouch (default C) to activate. To get down disable option then activate
      the change witch crouch (C).

      - To use Super Zoom, press NUMPAD 7 to enable option then use the Z key to 
      zoom in and X key to zoom out.
      
      - When Freeze AI is turned on the computer will still continue doing     
      whatever it was doing before it was frozen, so for example if you 
      were being shot at by the enemy, freezing the AI means he'll just 
      keep on shooting in that direction. This is actually a Mass Kill 
      option, because the animation isn't updated so you can still see the 
      AI actors moving and do things but they're actually considered      
      'dead', for example you can walk up to an officer and get his uniform 
      while he's smoking a cigar!
