     GENERAL GUIDE
     ###########################
     
     IGR (In Game Reset) works in two ways, like this:
     - RIGHT TRIGGER + LEFT TRIGGER + BACK + BLACK
     - RIGHT TRIGGER + LEFT TRIGGER + BACK + START
     
     Don't use XBOX Live when the MODchip is active, you will be banned.
     
     The DVD-ROM reads most burned discs HORIBLE, but DVD-R discs seems to work best.
     
     What the different partitions/drives is used to:
     - C: Operative System (DON'T TOUCH)
     - D: DVD-ROM
     - E: Chache, Saves, Music (IMPORTANT FILES)
     - F: Whatever you want (requires a harddisk bigger than 8 GB)
     - G: Whatever you want (for those who have formated a harddisk bigger than 120 GB in the OLD DAYS)
     - X: Chache
     - Y: Chache
     - Z: Chache
     
     Turn the machine off and on again once in a while, so it won't freeze, there is limited RAM (only 64 MB)
     
     Default username/password for the XBOX (FTP):
     - username: xbox
     - password: xbox
     
     Dasboards:
     - Avalaunch, UnleashX and Xbox Media Center is dashboards.
     - Every Dashboards has it's own network settings...
     
     Network setings:
     - DHCP: Automaticly reseves a IP-adress and all other network info
     - STATIC: Manually choose IP-adress and all other network info
     - IP: Manually chosen uniqe IP-adress (ex. 192.168.0.16) (Just make shure the last number is different on every component/pc)
     - SUBNET: Same at all components/pc (ex. 255.255.255.0)
     - GATEWAY: IP-adress to Server/rooter (ex. 192.168.0.1)
     - DNS1: IP-adress to server/rooter/pc that delivers Internet
     - DNS2: Alternative IP-adress to server/rooter/pc that delivers Internet
     
     Info about the filesystem:
     - Max lenght on files/folders is 42
     - Max files/folders per folder is 4096
     - Special symbols witch can't be used on files/folders is < > = ? : ; " * + , /  |
     - Special symbols witch can be used on files/folders is ! # $ % & ' ( ) - . @ [ ] ^ _ ` { } ~
