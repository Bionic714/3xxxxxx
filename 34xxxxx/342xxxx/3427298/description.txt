CalliGrapher 8.1 is the new version of popular handwriting recognition software for Windows Powered Pocket PCs. Most notable new features include support for Windows Mobile 5.0, a new soft input panel called Write Pad and the Statistical Analyzer. Additional new features include improved handwriting recognition engine, new color selector to choose pen and Write Pad colors, improved graphics, floating and attachable on-screen keyboard with autocomplete, improved support for landscape screen layout and full-VGA devices, and more.
