  Windows  XP  Embedded  is  the  operating  system and  development platform
  delivering  the power of the Windows operating system in componentized form
  to rapidly build reliable and advanced embedded devices.
  Based on the same binaries as Microsoft Windows XP Professional, Windows XP
  Embedded  enables developers  to choose from over 10,000 individual feature
  components  to  achieve  optimum  functionality  in  a  reduced  footprint.
  Versatile embedded-enabling features, such as  flexible  boot  and  storage
  options and headless support.
