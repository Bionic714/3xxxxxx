SRT to DVDit subtitle converter converts subtitles in SRT format to the format required by
Sonic's DVDit Pro (version 6 is what this program was made for). The program may be run
on any system that supports Java and has Java 5.0 installed. If you do not have Java 5.0 
(a.k.a Java 1.5 or Tiger) installed, you may download it at http://www.java.com.
