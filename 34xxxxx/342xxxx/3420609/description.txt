Around 340 C++ Projects for Visual C++
Some of the projects are in 6.0 (.dsw) 
some are (.vcproj)
simply extract the files using winrar
then burn / mount the iso image
with your favorite software.
if you have auto play enabled you should see a window come up
when you insert the cd or mount the image
if not run path to your cdrom lsd-autorun.exe
this will let you choose what programs to install
to browse the source code for all the projects go
to the /projects directory on the cdrom
