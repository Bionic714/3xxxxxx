There are many new features in Ultra-Sharpen 7.0 Pro all of which have been added because they make Ultra-Sharpen 7.0 Pro more powerful, more flexible and more user friendly.
* Full support for 16-bit images (Adobe Photoshop CS and Adobe Photoshop CS 2 only) as well as 8-bit images (Adobe Photoshop 6.0, Adobe Photoshop 7.0, Adobe Photoshop CS, Adobe Photoshop CS 2 and Adobe Photoshop Elements 3.0).
* Support for sharpening images in other color spaces like RGB, CMYK, Lab and Grayscale.
* Default settings for all actions are now user editable.*
* New and improved User's Guide.
* All new tutorial videos covering all of the actions, installation, modifying Ultra-Sharpen 7.0 Pro, batch sharpening, installing for Photoshop Elements 3.0, and more. Videos are in .AVI format which are playable on both Windows and Macintosh systems.
* Ultra-Sharpen 7.0 Pro supports Adobe Photoshop 6.0, Adobe Photoshop 7.0, Adobe Photoshop CS, Adobe Photoshop CS 2 and Adobe Photoshop Elements 3.0 on both Windows and Macintosh systems.
