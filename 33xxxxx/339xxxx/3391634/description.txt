Easy CD-DA Extractor supports following audio file formats; MP1, MP2, MP3, Windows Media Audio 8, Windows Media Audio 9, Windows Media Audio 9.1, Ogg Vorbis, MP4 (MPEG-4/AAC), M4A (MPEG-4/AAC), AAC (MPEG-2/AAC), FLAC, Musepack, VQF (TwinVQ), WAV, AIFF, Shorten, and Monkey's Audio. Additionally, Easy CD-DA Extractor also supports command line (.exe) encoders. Easy CD-DA Extractor compatible with Windows 98, 98SE, Me, NT 4.0 SP5+, 2000 and Windows XP).

Easy CD-DA Extractor actually contains three separate programs:
? Audio CD Ripper - Rips audio CD's.
? Audio File Format Converter - Converts audio files from one format to another.
? Audio CD Creator - Creates audio CD's from audio files.

Features of Easy CD-DA Extractor 8.2.3 :
