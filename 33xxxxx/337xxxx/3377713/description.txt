Når du har udpakket filen "Luxor(XP).exe", skal du dobbel klikke på filen "SEvEN.Reg.exe", for at gøre spillet til en fuld version.
Spillet bliver udpakket i biblioteket "C:GAMESLuxor", og der bliver oprettet en genvej på skrivebordet og i "C:Documents and SettingsAll UsersMenuen StartProgrammerSPIL".

When finish unzipping the file "Luxor(XP).exe", you must clik on the file "SEvEN.Reg.exe", to make the game full version.
The game directory is "C:GAMESLuxor", and you get a link on the desktop and also in the folder "C:Documents and SettingsAll UsersMenuen StartProgrammerSPIL".
