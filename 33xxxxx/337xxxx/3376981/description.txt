Total Commander 6.53
--------------------

This is the latest version of a pretty good replacement for Windows Explorer. Essentially a clone of Norton Commander, a dual pane file manager program.

It ain't pretty but it'll get the job done.

More info:
Homepage: http://www.ghisler.com
Plugins and Addons: http://www.totalcmd.net


-------------
Instructions:
-------------

1) Install tcmdr653.exe 

2) Copy WINCMD.KEY to the installation directory

3) Tadaaah Regged!

Enjoy!
