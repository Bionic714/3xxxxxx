FS2004 - FlyTampa Seattle-Tacoma (KSEA)

Installation:

1. create a folder inside the 'Addon Scenery' folder, named anything you, e.g. 'KSEA'

2. copy the 'scenery' and 'texture' folder to the 'KSEA' folder

3. rename 'FS2004SceneryWorldSceneryFL915150.BGL' to 'FL915150.XXX'

4. copy the 'FL915150_flytampa_mod.bgl' to that same directory

5. fly :-)
