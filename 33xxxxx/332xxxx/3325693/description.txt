El país de los juguetes. Versión 1.0 Fisher Price. Davison- Macromedia

El usuario puede elegir en la pantalla principal el acceso a seis actividades diferentes que están representadas por diferentes juguetes de Fisher-Price:-Super Taller:se realizan proyectos de carpintería usando los materiales y herramientas adecuados.-Super Garaje: se construyen coches a partir de diferentes piezas que se van combinando.-Super Cocina: se elaboran galletas a partir de una receta, después se decoran y se comen.-Combi circuito de Montaña:entre distintos vehículos hay que usar el más adecuado para entregar un paquete determinado.-La calle-escuela Habladora: encontramos números, letras, un reloj, animales, figuras colores, notas musicales… y pulsando sobre ellos se puede oír el nombre o el sonido, también ocurren animaciones.-Mesa para Artistas 3 en 1: en ella es posible escoger actividades de pintar, dibujar, construir con bloques o hacer manualidades.Algunas de estas actividades tienen dos niveles, con ayuda o de manera autónoma.
