Release-Notes:
==============

Total Commander (formerly known as Windows Commander) is a replacement for Microsoftâ€™s Windows Explorer. It
basically offers the same functionality, i.e. the usual features to copy, move or delete files, and on top
of that it can compare files, find duplicates and synchronize directories.
Besides that Total Commander (Publisher: Christian Ghisler ) also lets you handle archives
(ZIP, ARJ, LZH, RAR, UC2, TAR, GZ, CAB, ACE), rename groups of files and encode (and of course decode) files
in UUE, XXE and MIME format. The file manager also offers:

- Two panel interface
- FTP-client
- Drop-and-drag functionality
- Multilingual support
- Built in file viewer
- Split/Combine files

Features in Total Commander 6.51 now include :

* Two file windows side by side
* Multiple language support
* Enhanced search function
* Compare files / synchronize directories
* Quick View panel with bitmap display
* ZIP, ARJ, LZH, RAR, UC2, TAR, GZ, CAB, ACE archive handling + plugins
* Built-in FTP client with FXP (server to server) and HTTP proxy support
* Parallel port link, multi-rename tool
* Tabbed interface, regular expressions, history+favorites buttons
* New: Thumbnails view, custom columns, enhanced search
* And many more!
