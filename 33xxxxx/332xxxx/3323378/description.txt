Unreal Tournament GOTY (version 1999)

This archive contains the full version
Unreal Tournament GOTY.
No CD is needed and online-play is enabled.
Get the 4 Bonuspacks from the official site.


Installation:

- Extract Unreal Tournament.exe (self-extracting archive)
- Run the UTPatch436.exe (Patch to the latest 436 version)
- Replace the UnrealTournament.exe in /UnrealTournament/system/ with the one supplied
- Get the 4 Bonuspacks from the official site (recommended)
