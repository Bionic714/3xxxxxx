<p>Beyond Compare is a powerful time-saving utility for comparing files and folders on your Windows system.</p>

<p>Use it to manage source code, keep folders in sync, compare program output, and validate copies of your data. 
Beyond Compare can create a snapshot of a live directory structure, and compare it against the live copy at a later date.</p>

<p>Beyond Compare helps you analyze differences in detail and carefully reconcile them. 

It commands a wide range of file and text operations, as well as script commands for automating tasks.</p>
<ul>
  <li>  compare text files
  <li>  compare folders, zip archives, ftp sites
  <li>  update websites
  <li>  merge code changes
  <li>  compare program output
  <li>  and more
  </li>
</ul>

<p>Beyond Compare is a directory compare utility and a file compare utility wrapped in one. Synchronize files between laptop and PC, home and office, etc. Compare directory trees side-by-side with mismatches highlighted. Display text files side-by-side with changes highlighted. Great for merging changes to source code. Supports minor editing, Unicode text files, and Delphi form files. Save favorite sessions for quick access. Built-in FTP and zip support.</p>
