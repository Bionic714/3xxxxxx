Ethereal is a network protocol analyzer for Unix and Windows. It allows you to examine data from a live network or from a capture file on disk. You can interactively browse the capture data, viewing summary and detail information for each packet.

    * Development Status: 4 - Beta
    * Intended Audience: Developers, System Administrators, Education, Information Technology, Telecommunications Industry
    * License: GNU General Public License (GPL)
    * Operating System: All 32-bit MS Windows (95/98/NT/2000/XP), All BSD Platforms (FreeBSD/NetBSD/OpenBSD/Apple Mac OS X), All POSIX (Linux/BSD/UNIX-like OSes), Linux, OS X, Solaris, IBM AIX
    * Programming Language: C
    * Topic: Monitoring
    * Translations: English
    * User Interface: Win32 (MS Windows), X Window System (X11)
