	   Q. I want to add some scripts and maybe modify what's on that 
	      little popup thingy on the bottom (Sub-Menu) how how how?

	   Q. I'm using this so called oober xml u guys made... why is it 
              when I go into Videos, Music, Pictures that I hang for a bit 
              first?
	   A. Have a look at the xml see these tags:

	      <default>smb://xbox:xbox@192.168.0.2/xserver/movies/</default> 

	      correct them or if you don't know how then just make all 3

			   <default></default>  

	      (You really should learn to set smb correctly though, it goes 
              far in making your XBMC install tweaked to your network...

	      just for the other dashes I use. How do I make it so I can get
	      to them easily?

	      in there start making subfolders and putting other dashes in
	      them like ... C:Alt_DashesAvalaunchdefault.xbe. You will be
	      able to access them from My Programs by choosing Dashboards
