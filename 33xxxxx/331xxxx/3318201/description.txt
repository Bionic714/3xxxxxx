With this little handy program, you can play any game you want, using the lan connection in the game.

- Hoster -
1. Insert the ip's to the other players and click add
2. Click "Start" and run any game of your choice
3. When you're in the game, host a lan game.
4. Play

- Player -
1. Insert host ip, click add
2. Click "start" and run the game
3. "Find lan games" join the server that pop up
4. Play
