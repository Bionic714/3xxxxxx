vBulletin:

vBulletin ist eine herausragende Communitylösung, die Ihnen unkompliziert ermöglicht, ein Forum in Ihre Website zu integrieren. Dabei stehen Ihnen zahlreiche Funktionen zur Benutzerverwaltung, Konfiguration und Anpassung des Forums an Ihre Website zur Verfügung.
Entwickelt in PHP und gestützt durch das Datenbanksystem MySQL, bietet vBulletin optimale Geschwindigkeit und Stabilität auch bei hoher Beanspruchung.
vBulletin  Features

vBulletin Features:

vBulletin hat viele Features, die es im Vergleich zu anderen Forensystemen einzigartig machen:

    * Sehr schnelles MySQL Datenbank Backend
    * Komplett datenbankbasiert - Alle Informationen, Benutzer, Foren, Themen, Beiträge u.s.w. werden in der Datenbank gespeichert. Probleme und Unklarheiten bei Dateirechten oder Sorgen über die Sicherheit der Dateien sind Vergangenheit.
    * Hohe Skalierbarkeit - Der Datenbankserver und der Webserver können auf verschiedenen Rechnern laufen. Sie können sogar den Datenbankserver unter Unix und den Webserver unter Windows laufen lassen, wenn Sie das wünschen.
    * Jedes Detail der Oberfläche kann mit dem integrierten Templatesystem an Ihre Website angepasst werden.
    * Mit Hilfe der Importfunktion können Sie Daten eines anderen Forensystems einfach in vBulletin importieren. Diese Möglickeit besteht z.B. für Infopops Ultimate Bulletin Board (UBB.classic), UBB.threads, OpenTopic, ezboard, Ikonboard, PHPBB2 und noch einige mehr.
    * Eine leistungsstarke Suchmaschine
    * Unbegrenzte Anzahl an Foren, Beiträgen und Kategorien möglich
    * Vielseitige administrative Funktionen - Beiträge können mit einem einzigen Klick geändert, gelöscht, verschoben oder geschlossen werden!
    * Schutz gegen automatisierte Anmeldungen (Grafikbasierte Registrierungsüberprüfung)
    * Warteschlangen für Beiträge und Registrierungen - Überprüfen Sie neue Beiträge und entscheiden Sie, ob diese veröffentlicht werden oder nicht. Genauso können Sie neue Benutzer freischalten, bevor sie Beiträge schreiben dürfen.
    * E-Mail-Benachrichtigungen bei Antworten auf Beiträge
    * Unterstützung von Smileys, vB Code oder HTML in Beiträgen (vom Administrator konfigurierbar)
    * Templates - Administratoren können das Design des Boards mit dem intergrierten Templatesystem leicht ändern.
    * Benutzerprofile (änderbar vom Benutzer)
    * Benutzer aufgrund Ihrer IP-Adresse oder E-Mail-Adresse aussperren
    * Unbegrenzte Anzahl an Moderatoren pro Forum
    * Überprüfung der E-Mail-Adresse auf Echtheit während der Registrierung
    * Backups sind sehr leicht zu erstellen - Sie müssen nur eine einzige Textdatei herunterladen und haben schon ein komplettes Backup Ihres Forums.
    * Benutzergruppen - Geben Sie ausgewählten Moderatoren oder Benutzern erweiterte Rechte bis hin zu Administratorfähigkeiten.
    * Private Foren
    * Wortbasierte Zensurfunktion
    * Private Nachrichten - integriertes Nachrichtensystem für die Benutzer Ihres Forums
    * Beiträge seit dem letzten Besuch werden angezeigt, so dass Sie auf einen Blick sehen können, in welchem Forum neue Beiträge sind.
    * Konform mit dem Children's Online Privacy Protection Act (COPPA) von 1998.
    * Und noch viel, viel mehr...
