  Game Description

Pokémon Colosseum Bonus Disc is a promotional disc that features a preview of Pokémon Colosseum for GameCube as well as a sneak peek of the featurelength movie, Pokémon: Jirachi Wish Maker. Also on the disc is the exclusive character Jirachi, which can be added to the GBA versions of either Pokémon Red or Pokémon Sapphire via the Nintendo GameCube Game Boy Advance Cable.
