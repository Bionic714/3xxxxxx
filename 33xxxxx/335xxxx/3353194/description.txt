SLAX KB is a pocket operating system with the ability to run many Windows applications natively in Linux. It contains KDE, wine, dosbox and qemu.

SLAX is available as an ISO image of the CD. It fits on small 8cm CD-R(W). Burn it with your favorite cd-writing software and then boot from the CD. You might want to check your BIOS to assure it's set to boot the CD prior the disc.
