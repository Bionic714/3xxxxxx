Ultra Video Joiner is a tool to help you join or merge AVI(Divx,XviD,MPEG4), MPEG I/II, DVD(.VOB), RealMedia(.RM,.RMVB), VCD(.DAT), ASF/ WMV files in one large file in various format. You can add an unlimited number of video files to the list and easily rearrange their order if needed ,then merge the videos files to a large movie file in AVI(Divx,XviD,MPEG4), MPEG I/II, VCD, SVCD, DVD, WMV/ASF format. You even can use it to convert a single file.
