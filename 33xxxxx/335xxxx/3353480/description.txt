::: Dedicated server patch for Battlefield 2, release by Suspects. ::::

This is the V1.1 public/private/dedicated server patch-crak to be able to
accept newly upgraded V1.01 gameclients without a valid key.

For this to work you need to either have the V.1.1 server installed, DO NOT install this over the old V1.0 stand alone server, that wont work.

The installation of the V.1.1 server can be found here:
http://thepiratebay.org/details.php?id=3352773

Help seed and spread this as much and as fast as possible, that way we will see more V.1.1 servers faster.
