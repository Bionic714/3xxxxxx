Quake 3 
Start trek Voyager: Elite Force 
Soldier of Fortune II: Double Helix 
Jedi Knight II: Jedi Outcast 
Jedi Knight Jedi Academy

Doom 3 (yes!) 

These applications were made by Eridan With a little help from the rest of the MacRadiant community: http://www.redsaurus.net/gtkradiant/

-Mac OS X 10.3 (Panther) or later (Tiger 10.4 included) 
-Apple's X11 (can be found in the panther install disk 3 or: http://www.versiontracker.com/dyn/moreinfo/macosx/17747 )
