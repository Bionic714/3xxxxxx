Daemon Tools  is a disk image emulator and optical disc authoring program for Microsoft Windows.

The Lite edition only supports mounting images and is limited to 4 SCSI CD/DVD-ROM virtual drives.

As of December 2007, the following image formats are supported:[1]

    * b5t (BlindWrite 5)
    * b6t (BlindWrite 6)
    * bwt (BlindRead)
    * ccd (CloneCD)
    * cdi (Discjuggler)
    * cue (Cue Sheet)
    * iso (Standard ISO)
    * mds (Media Descriptor File)
    * nrg (Nero)
    * pdi (Instant CD/DVD)
    * isz (Compressed ISO)
