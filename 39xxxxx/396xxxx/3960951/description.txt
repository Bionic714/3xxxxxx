This is the Swedish language pack for Windows Vista ULTIMATE ONLY!!!


Installation Instructions!

1. Install the .msu file! (a hotfix that microsoft released some time ago!)

2. Start the computer in safe mode (f8 when you boot it)and install the .exe file there. (Thats the language pack!)

3. Restart the computer and in the controlpanel you select the Swedish Language Pack!

4. It's done!
