Toutou Linux est une distribution complète qui fonctionne sous Linux.

Toutou Linux est une mini distribution Linux en live-CD, installable sur disque dur.
son gros avantage est le chargement complet en mémoire, de ce fait elle est très rapides et libèrent le lecteur de CD/DVD pour son utilisation.
Elle possède tous les programmes utiles pour un usage familial : Traitement de texte, tableur, dessins, retouche d'images, lecteur multimédia, Internet, réseau, tchat etc...
C'est une distribution très intuitive.
Vous allez découvrir Linux et l'utiliser intuitivement.
Insérez le CD de Toutou Linux dans votre lecteur, et en moins de 5 minutes, votre ordinateur est devenu une bombe, vous avez accès à tout le contenu de votre disque dur sans rien altérer.

- Version française basée sur Puppy Linux 3.01


- Rox 2.6.1, nouvelle version du gestionnaire de fichiers interne à Toutou.
- Abiword 2.45, un traitement de texte plus léger mais plus pratique que MS Word. Il lit et écrit les fichiers .doc.
- Gnumeric 1.6.3, un tableur un peu moins puissant que MS Excel, mais lui aussi suffit largement pour tous vos calculs.
- Seamonkey 1.1.6, la suite Internet (navigateur Internet, client mail, news, carnet d'adresses, éditeur html).
- Grisbi 0.5.9, pour gérer vos finances personnelles.
- Inskapelite, un logiciel de dessins allégé. Tout est facile avec lui.
- Mtpaint 3.19, pour retoucher les images.
- Ical, un calendrier avec gestion des R.d.V.
- Gxine 0.5.9, pour regarder vos CD et DVD, écouter vos mp3.
- Gaim 1.5, pour tchater.
- GTKsee 0.6.0-beta, un clone d'ACDSee sous Winchoz pour visualiser vos photos.
- Grafburn 0.10.1, pour graver, effacer vos CD/DVD, l'équivalent de Nero
- Pbackup 3.1, un utilitaire de sauvegardes automatisées très élaboré.
- SoxGui, pour traiter vos fichiers sons.
- PupDVDTool 0.8, pour ripper, extraire, convertir, multiplexer etc... vos DVD.
- PBcdripper 3.0, un "ripper" de CD.
- PFind 3.3, outil de recherche de fichiers.
- Pdrive0.24, gestionnaire de montage avec détection automatique des médias amovibles.
- ePDFview 0.1.6, pour afficher les fichies PDF. Plus un utilitaire pour éditer des fichiers PDF.
- Pudd, un utilitaire permettant de copier un disque ou une partition.
- Gparted, un utilitaire pour gérer vos partitions.
- Pwireless 0.72 (scanner wifi).
- Pschedule 0.4 (un planificateur de tâches).
- Pprocess 0.2 (un gestionnaire des processus actifs).
Un nouveau fichier de cloture de session permettant le choix, lors du prochain démarrage et pour l'accélérer, d'ignorer ou non vos dispositifs série.

Et plein d'autres logiciels utilitaires, outils multimédia et de paramétrages. Et bien sur des jeux.
