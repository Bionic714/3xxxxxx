FakeTwin 2.0


FakeTwin is a program, with which you can copy a complete existing folder, while the copied files will be fake. All copied folder and file names will be the same as in the source but the files will either be empty or they will be filled with spaces. 

I personaly use it for replacing annoying custom sounds in steam related games, works like a charm.
