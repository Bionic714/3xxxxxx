3 Handy Webmaster Applications for Windows from Harmony Hollow Software

This bundle includes Cool Web Scrollbars, Javascript Menu Master, and IE CountKeywords.

Cool Web Scrollbars allows you to quickly and easily generate the CSS code needed to have your website's scrollbars colorized to match the rest of your site's colors when it is viewed with Internet Explorer.

Javascript Menu Master lets you easily create navigational menus for your site with no programming knowledge.

IE CountKeywords is a browser add-on for Internet Explorer that adds a new item to IE's context menu allowing you to right-click on any webpage, select the item, enter a keyword or phrase, and then counts the occurrences of that keyword or phrase in the text of the page. Very useful for search engine optimization.
