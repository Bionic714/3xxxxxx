This app reads web pages, text, or other sources into mp3 files using voices. 
Pro mode will process lists of files. Retains list of source files.
Auto splits the mp3 files into smaller sequentially-numbered mp3s. Also has unzip, file rename, and unrar. Adds itself to context menu SendTo in win explorer.  SAPI5.1 and/or SAPI4.   www.ArkansasNetsoft.com for more info
