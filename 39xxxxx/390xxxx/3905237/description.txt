Manual:
1)Install Celestia
2)Start Celestia, change settings...
3)Mount iso image - add this line to /etc/fstab:
/path/to/Celestia_Extras_SolarSystemV1.iso /somepath/Celestia_Extras   iso9660 noexec,nosuid,nodev,loop        0       0
Windows users can use Daemon tools
4)Make symbolic link for SolarSystemExtras in extras folder.  in my system it is /usr/share/celestia/extras
check:
/usr/share/celestia
/usr/local/share/celestia
/opt/Celestia  and
/opt/CelestiaCVS
C:\program filescelestia
E:\program filescelestia
5)Restart Celestia and have fun!
