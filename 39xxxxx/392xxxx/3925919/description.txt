Changer XP is an integrated tool to personalize the IE Skin, the Win XP 
    logon screen, the desktop wallpaper,  the screen saver as well as the 
    boot screen  of Windows 9x/Me/2000/XP all in one. 
    
    With Changer XP, there would be little trouble but fun to set  your favorite 
    images (BMP, JPG, GIF, PNG, TIFF, TGA and WMF format are 
    supported) as your boot screen, desktop wallpaper and IE Skin. Besides, 
    the screen saver programs can run in the order you like. With only clicks, 
    you can get a unique boot screen each time you boot up Windows.
