EASYBCD is NeoSmart Technologies' multiple award-winning bootloader
management tool for Windows Vista. It super-powers Vista's default
bootloader, adding support for all previous versions of Windows, 
along with Linux, OS X, BSD, SkyOS, ReactOS, Solaris, and more.

EasyBCD is freeware.

With EasyBCD, you can add and remove entries, change timeouts, rename
existing entries, configure advanced options, limit the amount of 
memory available to Windows, enable loading of unsigned drivers in 
Windows Vista x64, debug the boot process, reinstall the Vista
bootloader, fix a corrupted bootloader and more!
