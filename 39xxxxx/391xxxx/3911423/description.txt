AlienGUIse Theme Manager + All current Alienware Suites

Suites: AlienMorph, Alienware Area-51 Superman(TM), ALXMorph, Darkstar, 
Invader, Star Wars - Dark Side, Star Wars - Light Side & XenoMorph.

All suites contain a Windows Media Player skin, a complete customized 
icon set, a custom Windows desktop theme and a wallpaper.

Install: Run AlienGUIse_Setup.exe and extract Themes.7z (compressed with 
7-Zip - www.7-zip.org) to %ProgramFiles% - AlienGUIse - Themes.
Run AlienGUIse, select one of the Suites and click on Apply.
