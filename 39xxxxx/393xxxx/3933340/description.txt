1. What is XtraTools 2008?

XtraTools 2008 - a set of applications consisting of more than 
20 independent system utilities that are complimentary to the 
standard MS Windows utilities. XtraTools can increase productivity 
and makes working with MS Windows easier.

XtraTools 2008 contain tools:

WinTools.net - suite of tools for increasing MS Windows operating 
system performance.

Armor Tools - suite of tools which ensure your privacy and keep 
sensitive information secure.

Extra Drive Pro - suite of tools for creating Extra Drives on 
your PC (physical drives remain unchanged).

RAM Saver Pro - easy-to-use RAM optimizer tool that will keep your 
computer running faster.

The Fast File Encryptor - can encrypt or decrypt files, contents 
of folders and contains a lot of additional tools.


System requirements. 
OS MS Windows 2000/XP
Minimum screen resolution 800x600 
Pentium processor
10 Mb free disk space 


Attention! To use this program you need the administrator`s rights. 
Before working with the WinTools.net we recommend that you close 
all other applications and create registry backup. 


2. Features

XtraTools 2008 features the following tools:

WinTools.net - a suite of tools for increasing MS Windows operating 
system performance. WinTools.net cleanly removes unwanted software 
from disk drives and dead references from the Windows registry. 
WinTools.net puts you in control of the Windows start up process, 
memory monitoring and gives you the power to customize desktop and 
system settings to fit your needs. 

Armor Tools - a suite of tools to ensure your privacy and keep your 
sensitive information secure. Armor Tools makes it easy for you to 
remove traces of your activities from the computer - deleting 
documents, programs and other files so that they can not be recovered. 
Armor Tools also features tools for making additional safety related 
adjustments including five independed tools...

Extra Drive Pro - a suite of tools for creating Extra drives on your 
computer (physical drives remain unchanged and are not divided into 
additional partitions). The program includes tools for creating 
Encrypted drives (he drive is encrypted by 256-bit / 32 - symbols) 
password using "AES256" algorithm in real-time mode), Virtual CD-ROMs, 
Virtual Drives based on files, Virtual Drives based on folders, 
RAM-drives...

RAM Saver Pro - an easy-to-use RAM optimizer tool that will keep your 
computer running faster. It increases the operation system performance 
by making more memory available for your applications. "RAM Saver Pro" 
will: defragment system memory for faster access time; recover memory 
leaks from poorly behaved applications; temporarily flush unused 
libraries out to disk and so on.

The Fast File Encryptor - is designed for fast processing of extremely
large files. It can encrypt or decrypt files, contents of folders and
contains a lot of additional tools. The Fast File Encryptor presents 
a flexible user interface with detailed messages about all operations.
