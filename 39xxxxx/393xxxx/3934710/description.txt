Update your wii to use the latest configs and to be able to play the latest games:)



1. UNRAR the file using winRAR
2. UNRAR the update you want to use
3. Burn the ISO with IMGBURN at 4x speed using a DVD-R or a DVD-RW



note: it will first test if your system is able to get updated. when it is done then you press A or RESET again to update your wiikey:)
