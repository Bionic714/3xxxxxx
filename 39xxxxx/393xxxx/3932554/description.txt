*********************************
*             Setup             *
*********************************

First extract the map extrator (either enUS or enGB depending on wow version) to your wow directory.

Then run ad.exe

When it has finished place the maps folder into C:/AC Web Ultimate Repack/ascent

Then start up the control panel (C:/AC Web Ultimate Repack/Control Panel.exe)

Click the "Launch Server" Button.

Now change your realmlist (realmlist.wtf located in your wow directory) to 127.0.0.1

login with user: admin pass: admin

Now play!

*********************************
*           Website             *
*********************************

To get the website to work publically, open 
C:/AC Web Ultimate Repack/Server/htdocs/configuration.php with notepad, 
search for live, and replace http://localhost with your site. 
Also you will need to go into the forum admin cp and change the board directory from http://localhost to your site.
The Joomla control panel is found at http://localhost/administrator, user=admin pass=admin
