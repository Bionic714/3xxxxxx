POI FACTORY REDLIGHT CAMERA and SPEED CAMERA POI FILES FOR THE UNITED STATES.   
----------------------------------------------------------------------------
Updated January 09 2008

This is a POI File for use in the united states.  It will work without modification on All Garmin GPS units,
and SHOULD work on other units also.  Some format modification may be required for other brands of GPS.

It contains the location of 2733 Redlight Cameras, and 114 Speed Cameras.

Description of files included.

Redlight-Cameras.csv 		 -- Redlight Camera ONLY POI file (Redlight Camera POI Database)
Redlight-Cameras.bmp             -- Generic icon of a Red Light

Speed-Cameras.csv   		 -- Speed Camera ONLY POI File    (Speed Camera POI Database)
Speed-Cameras.bmp                -- Generic icon of a Police car

USAmap.jpg                       -- Areas Covered

Extras\Icons         		 -- (Not Required) Custom ICONS For use on Garmin Units.   
Extras\Alerts      		 -- (Not Required) Custom Alert Sounds for use on Garmin Units.


To upload the units to your garmin unit, Use the Garmin POI loader software available from garmin- 
http://www8.garmin.com/products/poiloader/

For the Redlight-Camera.csv, If you choose to do a "Manual Install", you can choose to be Alerted--
	 "When ever you get close to a point within xxx feet".  I go with 1320 Feet.



If you want to get creative, Go to the extras folder and you can pick your own Icons and Alert sounds.

CUstom Icons:  Same thing, just overwrite the Redlight-Cameras.bmp or Speed-Cameras.bmp file with your own, then upload using POILOADER.
Custom Alerts: If your GPS Supports mp3 playback, copy your alert sound from the Extras\Alerts folder to the main directory.
	     You MUST give it the same name as the type of alert.  Such as Redlight-Cameras.mp3 will alert for Redlight cams,
    	     and Speed-Cameras.mp3 will alert for speed cams.
